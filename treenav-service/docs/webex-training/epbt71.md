# Set Up and Schedule a Webex Training Session

## Choose a Level of Security for a Training Session

You provide security for your training session by doing any of the following: 

- Require a password: Most Webex Training websites require you to specify a password for any session that you host. Attendees must provide this password to join the session.
- Specify an unlisted training session: You can specify that the session is unlisted. An unlisted training session doesn't appear on the training session calendar. To join an unlisted training session, attendees must provide a unique session number.
- Require attendees to sign in with single sign-on authentication before joining this session: You can restrict attendees to those who were directly invited, preventing other people joining from a forwarded invitation.
- Restrict session to invited attendees only (registration options will be disabled): You can restrict the session to invited attendees only. Attendees can't forward the invitation.
- Exclude the session password from email invitations: You can prevent the password from appearing in the email invitations that your Webex training website automatically sends to attendees.
- Require attendees to log in: You can require that attendees have a user account on your Webex training website. Thus, attendees must sign in to your site before they can attend the session.
- Require attendees to register for the training session: You can require each attendee to fill out a registration request that includes their name, email address, and other information, and then send the request to you. You can then accept or reject each registration request.
- Restrict access to the training session: Once all attendees have joined a training session, you can prevent more attendees from joining it by restricting access to the training session.  
Choose a level of security based on the training session's purpose. For example, if you schedule a session to discuss your company picnic, you can probably specify just a password for the session. If you schedule a session in which you discuss sensitive financial data, you may want to specify that the session is unlisted. You may also choose to restrict access to the session once all attendees have joined it.

## Set Up a Topic Name and Password

When scheduling a training session, you set up the topic name and password. You can edit the topic name and password on the Edit Scheduled Training Session page. 

A training session password should contain a minimum of 4 characters, and can contain a maximum of 16 characters. A password can't contain the following characters: \\ \` " / & < > = \[ \], and can't be your username, the host's name, the topic name, or the site name.

| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Specify a topic for the training session in the Topic box. You can enter a password in the Set Session Password field.                                                                                                                           |
| 3 | Specify your options, and then select Schedule or Update.                                                                                                                                                                                        |

## Set Up a Large Training Session with Over 500 Attendees

You can set up a large training session by allowing over 500 attendees. However, if you choose this option, some Webex Training features are limited or aren't supported. These include: 

- The multipoint video, private chat, and active talker visual cue features aren't available.
- Attendees can't send video and view the names of other attendees.
- Each Breakout session and Hands-on Lab computer supports a maximum of 100 attendees.

 You can't change this option within the session. 

| 1 | On the Schedule Training Session or Edit Scheduled Training Session page, scroll to Session and Access Information.              |
| - | -------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Check the This session will have over 500 attendees check box. A message box appears, describing the limitations of this option. |
| 3 | Select OK. If you schedule large training sessions frequently, select Save As Template at the bottom of the page.                |
| 4 | If you have completed specifying options, select Schedule or Update.                                                             |

## Copy Session Information from Another Session

You can copy specific session information from other live sessions that you scheduled, or from sessions that were scheduled on your behalf. Some scheduling information varies from session to session, and therefore can't be copied into the scheduler, including: 

- Date, time, and recurrence (except duration, time zone, and join-before-host options)
- Invited attendees and presenters lists (except estimated number of attendees and presenters)
- Registration data of registered attendees
- Hands-on lab information and reservation
- Course material and tests

| 1 | On the Schedule Training Session or Edit Scheduled Training Session page, scroll to Session and Access Information and select Copy from. |
| - | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the session from which to copy session information and select OK.                                                                 |

## Use the Scheduling Templates

When scheduling a training session, you can save scheduling option settings as a template for future use. Some scheduling information varies from session to session, and therefore cannot be imported into a template. The following scheduling options are not passed on to the template: 

- Date, time, and recurrence (except duration, time zone, and join-before-host options)
- Invited attendees and presenters lists (except estimated number of attendees and presenters)
- Registration data of registered attendees
- Hands-on lab information and reservation
- Course material and tests  
When you choose a template, the scheduler's option fields are populated with the values for the chosen template. Each template can be updated or saved as a new template on your Webex Training website.

| 1 | On the Schedule Training Session page or Edit Scheduled Training Session page, enter data into the scheduling option fields or change settings.                                                                                             |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Save As Template, located at the bottom of the page.                                                                                                                                                                                 |
| 3 | Fill in the Template name field, or change the name if modifying an existing template.                                                                                                                                                      |
| 4 | Select Save. The next time you schedule a session, you can select your template from the Set options using template list. To delete a template, select My Webex > My Profile > Scheduling Templates. Select the template and select Delete. |
