# Add Numbers to a Customer's Existing Devices

Whether they're in the middle of a trial and have access to 10 free PSTN numbers or have been converted to a paid subscription, all you have to do is follow a few steps to enable calling on their devices. 

Before you begin

- PSTN on Cisco IP phones and Webex desk and room devices is only available in the United States.
- PSTN for Webex desk and room devices does not require a Webex Calling license.

| 1 | From the partner view in <https://admin.webex.com>, go to customers and select a customer.                                                                                       |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Call \> Add Numbers                                                                                                                                                       |
| 3 | Enter the customer's PSTN contract information.                                                                                                                                  |
| 4 | Enter an emergency service address for the customer. The PSTN service provider instantly validates the address.                                                                  |
| 5 | Search for and add standard numbers.                                                                                                                                             |
| 6 | Select Next.                                                                                                                                                                     |
| 7 | Review the order, then click Place Order.                                                                                                                                        |
| 8 | Select Set Up Cisco Webex Service Customer has to accept the order before you can proceed to the next step.                                                                      |
| 9 | Follow the steps in the setup wizard for Plan Review, Call Settings (this is where you fill in PSTN information), Message Settings, Enterprise Settings, Add Users, then Finish. |
