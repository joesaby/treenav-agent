# Control Hub application status alarm definitions

__Table 1\. Threshold legend__
| Threshold value                | Definition                                                                                                                                         |
| ------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0                              | Notification is triggered by UC Apps default threshold value.                                                                                      |
| N/A                            | There’s no significance for threshold here. All such events are notified. (_Example_: Insufficient Licenses, Unified CM Service failure and so on) |
| Specific numbers like 10/60/90 | The threshold value is set for the notification to be triggered.                                                                                   |

We monitor ungraceful shutdown notifications on Dedicated Instance applications and will automatically clear the notifications after validating the applications. A support case isn’t required to address these notifications.

We periodically monitor and fix the cross-launch failures on Dedicated Instance applications. A support case isn’t required to address these notifications.
