| 1 | In a space, hover on a message and click ![More Options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/458001-459000/458469.jpg) More Actions \> Pin Message.                                                                                                                                                                                                                                                                                                                     |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | (Optional) Select Pin message as banner. Message appears as a banner at the top of the space.                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| 3 | Click Pin. To see all the pinned messages, click View pinned messages![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/458001-459000/458904.jpg). Pinned messages are ordered by the date they were pinned. You can also unpin a message by:clicking the pin icon on the message.clicking View pinned messages![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/458001-459000/458904.jpg), select the message in the list, and click Unpin. |

| 1 | In a space, long press a message then select Pin.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Tap Pin or Pin message as banner. To see all pinned messages in the space, tap ![More options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/461001-462000/461853.jpg), and then tap Pins. To unpin a message from the list, tap Edit, select the message, and then tap Unpin. To make it easier to view pinned messages on a smaller screen, we prioritize showing the message content, or the file name. On large-screen devices like iPads, you'll see the message content, followed by the sender's name, date, and the time. When a pinned message is no longer available, due to your organization retention policy, it will be removed. |
