# Hand off interpreting to the next interpreter in Webex meetings and webinars

Each interpreter for your assigned language gets moved to the same audio channel so that you can hear each other as well as the original meeting audio. When it's time for the next interpreter to begin, hand off interpretation duties to them so ensure a smooth transition for attendees.

The host determines the order of interpreters. The host can also manage the order in the meeting. Interpreter 1 can hand off to interpreter 2, then interpreter 3, then back to interpreter 1, and so on.

This feature is supported on Webex Meetings and Webex Webinars. On Webex App, this feature is available on our Webex Suite meeting platform. [Find out if your meetings use the Webex Suite meeting platform](https://help.webex.com/m61d8eb/).

On Webex App, this feature isn't available for Webex for Government.
