# Enable the Webex Meetings Desktop App in Microsoft Outlook (Windows)

## Check Your Cisco Webex Meetings Desktop App Preferences

| 1 | From the Windows Start menu, open the Cisco Webex Meetings desktop app.                                                                        |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the gear ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/410001-420000/417001-418000/417392.jpg) icon. |
| 3 | From the drop-down list, select Preferences.                                                                                                   |
| 4 | Select the Integrations tab.                                                                                                                   |
| 5 | Make sure that the Microsoft Outlook check box is checked.                                                                                     |

## Check Your Microsoft Outlook Preferences

| 1 | From Microsoft Outlook, go to File \> Options \> Add-ins.                                 |
| - | ----------------------------------------------------------------------------------------- |
| 2 | At the bottom of the pop-up window, in the Manage: drop-down list, select Disabled Items. |
| 3 | Select Go....                                                                             |
| 4 | In the Disabled Items dialog box, select Add-in: WebExOI.Addin (ptolkadd.dll).            |
| 5 | Select Enable and Close.                                                                  |
