# Create and manage line key templates

## Feature specifications and caveats for line key templates

A template is a bundle of settings that are model-specific. Templates can be built and applied on an organization, location, or device level and applied in bulk. Templates allow for uniform configuration to specific device models. Once you configure the settings, you can sync the phones immediately or schedule a later time to sync and apply the settings. 

After a template is applied, you can customize the buttons for individual devices, if needed. For more information on how to configure layouts for individual devices, see [Configure layouts for Cisco phones in Control Hub](https://help.webex.com/article/nb0brcp/).

Users can't configure their device line key assignments.

Bulk application isn't available for Key Expansion Modules (KEMs). You can only customize buttons on an individual KEM by going to Devices \> Configure Layout.

### Configurable line key options

- Primary line key \- This is the user’s primary extension, which can appear multiple times on the phone but the very first appearance is the user’s primary extension.
- Shared line/virtual line key \- This shows the appearance of other users on the owner’s phone.
- Monitor keys \- These keys are dedicated to user and call park monitoring. Monitor keys are configured from the top of the configured monitor list.
- Speed dial key \- A speed dial label and value, these keys are configured by the administrator. Unlike other buttons on the template, the value is directly attached to the specified appearance of the line key.
- Open keys \- An open key automatically takes the configuration of a monitor button starting with the first open key. Users can configure these buttons for speed dial numbers.
- Closed keys \- Button not usable but reserved for future features.

The first button on a device must be the primary line.

## Create a line key template

Line keys are the buttons next to the device display. You can program line keys with primary, shared lines, or calling functions. Administrators can create line key templates, by device model, to bulk assign phone layouts. Use these steps to create line key templates at the organization level.

When creating a template for Cisco Desk Phone 9871, consider the possible scenarios where the phone may not have attached KEMs, or may have attached KEMs with configured lines, and ensure that you appropriately create, name, and apply a template.

| 1  | Sign in to [Control Hub](https://admin.webex.com).                                                                                                                                                                                                                                                                                                                                                   |
| -- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2  | Go to Services \> Calling.                                                                                                                                                                                                                                                                                                                                                                           |
| 3  | Click Service Settings.                                                                                                                                                                                                                                                                                                                                                                              |
| 4  | Scroll to Devices and click View Templates.                                                                                                                                                                                                                                                                                                                                                          |
| 5  | Click Create custom template.                                                                                                                                                                                                                                                                                                                                                                        |
| 6  | Enter a unique name for the line key template, select the device model, and click Next.                                                                                                                                                                                                                                                                                                              |
| 7  | Assign Line Key Assignments in the provided fields. Depending on your phone’s existing line keys, you can assign up to ten line keys. Assignments such as shared lines and monitored lines are based on the values you set within Control Hub.                                                                                                                                                       |
| 8  | If assigning a Shared / Virtual line, choose the index number from the drop-down. The index number is the line order number of the shared or virtual line assigned in the configured lines. As you can add multiple appearances of the same shared or virtual line on a phone, choosing the index number assigns the respective line to a line key based on the assignments in the configured lines. |
| 9  | (Optional) If creating speed dial entries, enter the Line Key Label, and then the Destination phone number or extension.                                                                                                                                                                                                                                                                             |
| 10 | Click Next and review the updates that are made to the template.                                                                                                                                                                                                                                                                                                                                     |
| 11 | Click Submit. A notification displays on the successful completion of the line key template.                                                                                                                                                                                                                                                                                                         |

## Reset a line key template

You can't delete or view the default template. You can only apply a template to reset selected devices to their default line key templates. This also overrides any custom line key settings.

| 1  | Sign in to [Control Hub](https://admin.webex.com).                                                                                                                                                                                                                                                                                                    |
| -- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2  | Go to Services and click Calling.                                                                                                                                                                                                                                                                                                                     |
| 3  | Click Service Settings.                                                                                                                                                                                                                                                                                                                               |
| 4  | Scroll to Device and click View Templates.                                                                                                                                                                                                                                                                                                            |
| 5  | Click Reset Line Key templates.                                                                                                                                                                                                                                                                                                                       |
| 6  | Select the devices using the Locations search. You can enter up to 10 locations.                                                                                                                                                                                                                                                                      |
| 7  | Select the Devices (by model) to apply the template to.                                                                                                                                                                                                                                                                                               |
| 8  | (Optional) Select the Refine search by Tags toggle to search using tags. For example: sales, accounting, or marketing. Tags are the labels that you give to your organization’s devices when onboarding in Control Hub.                                                                                                                               |
| 9  | (Optional) Select the Refine search by advisories toggle to search devices that had one or more advisories. You can specify primary line, shared line, monitoring line, and so on, in the settings. If there’s a mismatch between the settings and the device list, an advisory message appears on the screen. You can choose to ignore the advisory. |
| 10 | Click Next. A preview of the devices that match the search criteria displays.                                                                                                                                                                                                                                                                         |
| 11 | To apply the default template configuration, click Reset to default. The tasks page displays your configuration summary. The Results page lists a summary of the devices that the template is applied and the device where the advisory is displayed.                                                                                                 |

## Configure lines for a device

### Configure lines for a device with a custom layout

On configuring the device template, you can customize the layout for the user (shared lines, monitored lines, or speed dials). If there’s an unused or open line key, it’s automatically assigned as a configured monitored line to the previous setup within Control Hub. 

On completing the customization of your template, you can preview the layout by toggling Preview Layout. The result displays within the template window in Control Hub. 

If a template is applied, click Configure Layout to open the custom layout screen.

### Configure lines for a device without applying a template 

The phone is reverted to the basic layout or the default state with no customizations applied. The basic layout corresponds to the configurations in the Configure Lines modal, and from a user’s monitor list. The basic layout is read-only. 

### Situational banners

When configuring templates, situational information banners appear when previously configured features don’t sync with the assigned line key template.

Examples of situational information banners include: 

- There are more shared appearances than shared users.
- There are more shared users than appearances.
- There are more monitor appearances than monitors.
- The device isn’t configured to support the user’s monitor list.
