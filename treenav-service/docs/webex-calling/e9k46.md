# Webex App | Turn on hardware acceleration

| 1 | Click your [profile picture](https://help.webex.com/m67xht/), and then select Settings \> Video.                             |
| - | ---------------------------------------------------------------------------------------------------------------------------- |
| 2 | Check the Enable hardware acceleration check box. If your system doesn't support this feature, you won't see this check box. |
