# Integrate HubSpot with Webex

**IMPORTANT**: Cisco Webex announces the discontinuation of the Hubspot integration with Webex. This integration will no longer be supported, and no new installations will be possible beyond April 30th, 2024\. If you have the integration already enabled, then it should continue to work until July 31st, 2024.

The [Hubspot Bot](https://help.webex.com/en-us/article/dytwf3/Webex-App-%7C-HubSpot-bot#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5Fcbccbec5-8baf-4d26-b0e1-a60f9f03128f) is not affected by this announcement and will continue to be supported.

## Install Webex in HubSpot

HubSpot organization administrators can install Webex from the HubSpot marketplace. Once installed, all HubSpot users in your organization can connect to and start using Webex. Each user won't need to install Webex from the HubSpot marketplace.

You can connect a single Webex account to a HubSpot account.

Before you begin

Ensure that you have HubSpot app marketplace permissions.

| 1 | In HubSpot, click ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/464001-465000/464787.jpg) marketplace in the main navigation bar and then select App Marketplace. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Use the search bar to find and select the Webex integration.                                                                                                                                                       |
| 3 | Click Install app.                                                                                                                                                                                                 |
| 4 | Select the HubSpot account you want to use for the integration and click Choose Account.                                                                                                                           |
| 5 | When prompted to grant permission to Webex to access your HubSpot account, click Connect app.                                                                                                                      |
| 6 | Enter your Webex email address and click Sign In. Depending on your organization's configuration, you're redirected to another page to sign in using your SSO credentials.                                         |
| 7 | Enter your account details to sign in. Webex is now integrated with HubSpot and the HubSpot Accounts page appears. Your users can now start to use Webex to collaborate with leads and customers.                  |
