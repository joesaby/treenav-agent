# Upload files before a webinar

To learn more about turning this feature on for your Webex site, [contact your administrator](https://help.webex.com/7ksvf1/).

You can upload up to 20 files. Files must be smaller than 20MB, and filenames cannot contain special characters (`", \, /, :, ; *, ?, <, >, |, #`). 

Supported file types are Microsoft Excel, Word, and PowerPoint, .pdf, .csv, .txt, .jpg, .png, .gif, .svg, .mp4, and .mov.

| 1 | After [scheduling a webinar](https://help.webex.com/n2ox4qo/), select Calendar, then select it from your upcoming webinars.        |
| - | ---------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Webinar materials section, select Edit, then drop your files into the window or select Choose file to browse for the files. |
| 3 | Click OK.                                                                                                                          |
