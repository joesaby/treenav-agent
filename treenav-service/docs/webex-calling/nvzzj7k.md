# Upgrade a Webex Calling license for a workspace and user

## Upgrade a workspace's Webex Calling license

Upgrading the license maintains the existing settings of the workspace and gives access to additional Calling features. To find out more about the features that are available with the common area workspace and professional workspace licenses, see [Features available by license type for Webex Calling](https://help.webex.com/n1qbbp7).

Before you begin

- Once you've upgraded a workspace to the professional workspace license, you can't downgrade to a common area workspace license.
- After upgrading to the professional workspace license, if you want to switch to another subscription, the new subscription also takes the professional workspace license. You can't downgrade to a common area workspace license.
- Workspaces allow editing their licenses. If Microsoft Teams Rooms (MTR) is enabled for a workspace, a user can’t upgrade or downgrade from Basic licenses to Webex Calling licenses. Also, a user can’t enable the Webex Calling licenses as it is restricted, and the option isn’t visible unless it was previously assigned.

| 1 | Sign in to [Control Hub](https://admin.webex.com).                                                                                                                                                             |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Go to Management \> Workspaces, and choose a workspace to upgrade the license for.                                                                                                                             |
| 3 | Under the Calling tile, click the gear icon ![Settings button represented by a cog icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/438001-439000/438264.jpg). |
| 4 | Choose Cisco Webex Calling and click Next. If you have multiple subscriptions, all your subscriptions appear.                                                                                                  |
| 5 | Under the Calling service, select the Webex Calling check box, choose the Professional workspaceoption, and click Save.                                                                                        |

## Upgrade a user's Webex Calling license

If you upgrade a user's license to professional license, then you may need to [follow these steps](https://help.webex.com/article/9wqlsbb) to add more devices to that user. Clients and applications are automatically assigned to users you upgrade.

Find out more about features that are available with the standard and professional licenses in [this table](https://help.webex.com/en-us/article/n1qbbp7/Features-available-by-license-type-for-Webex-Calling).

Calling services become unavailable for users who run Webex Calling versions 44.3 or earlier.

Before you begin

You can't downgrade from a professional to a standard license without loss of all provisioning and configuration information for all affected users. We recommend this option to only be considered in extreme circumstances where all other options have been exhausted.

| 1 | Sign in to [Control Hub](https://admin.webex.com).                |
| - | ----------------------------------------------------------------- |
| 2 | Go to Users and select the User whose license you want to change. |
| 3 | Under Licences section, select Edit licences.                     |
| 4 | Under Calling, select the Professionalcheck box, and click Save.  |
