# Manage phone numbers in preferences for Webex Meetings

- [Windows](#id%5F119426)
- [Mac](#id%5F120670)

| 1 | Open the Meetings desktop app and sign in.                                                                                                                                                                                                                    |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the top right corner of the Meetings desktop app, select the App settings ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/410001-420000/417001-418000/417392.jpg) icon, then from the drop-down list, select Preferences. |
| 3 | For each preference you set, choose Apply to save your changes.                                                                                                                                                                                               |
| 4 | View Saved phone numbers and click Edit to change the saved phone numbers. You are taken to the Webex site where you can update and save the new numbers.                                                                                                     |
| 5 | Click Clear to delete the recent phone numbers.                                                                                                                                                                                                               |

| 1 | Open the Meetings desktop app and sign in.                                                                                                                                                                                                                                         |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | On Mac, select the Webex Meetings menu  \> Preferences \> Phone numbers                                                                                                                                                                                                            |
| 3 | For each preference you set, choose Apply to save your changes.                                                                                                                                                                                                                    |
| 4 | View Saved phone numbers and click Edit to change the saved phone numbers. You are taken to the Webex site where you can update and save the new numbers. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/439001-440000/439283.jpg) |
| 5 | Click Clear to delete the recent phone numbers.                                                                                                                                                                                                                                    |

To manage your meeting join preferences in advance, see [Set your Preferences in the Cisco Webex Meetings](https://help.webex.com/oqq5p4/).
