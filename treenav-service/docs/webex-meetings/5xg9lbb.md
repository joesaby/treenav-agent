# Change Language Settings for a Cisco Webex Site in Cisco Webex Site Administration

## Change the Default Language for a Site

You can change the default language of your Webex site, so that users who haven’t set their language preference see the site and meeting invitations in the default language.

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                      |
| - | --------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Site Options section, choose the default language from the Default language for the Webex site drop-down list. |
| 3 | Click Update.                                                                                                         |

## Change the Languages Available to Users

You can choose what languages are available to your users, in case your site doesn't support certain languages.

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                                                              |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Site Options section, go to the Languages available in Preferences, and check the check box next to each language you want to make available to users. |
| 3 | Click Update.                                                                                                                                                 |
