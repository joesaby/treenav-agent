# Webex App | Delete whiteboards

From within the app, use the following instructions to delete [whiteboards that you create in a space](https://help.webex.com/ne9qdcz/) or [whiteboards that you create during a meeting associated with a space](https://help.webex.com/vwot6eb/). You can go to [Webex Whiteboards](https://help.webex.com/up6k4ab/) to delete these, and any other whiteboard that you create.

- [Desktop](#task%5FBE40BC7ECC1987374342B4F665ADD3F6)
- [iOS](#id%5F125508)
- [Android](#task%5F1845D5E23831704475A464276B177B2E)

| Select the space, and click Content. Then select ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/443001-444000/443562.jpg) and click Delete ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/428001-429000/428484.jpg) on each whiteboard or annotation that you want to delete from the whiteboards list. |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| 1 | In the space, go to the activity menu ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/461001-462000/461853.jpg) and select Content \> Whiteboard. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Long press the whiteboard that you want to delete, and then tap Delete.                                                                                                                          |
