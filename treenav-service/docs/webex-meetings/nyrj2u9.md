# Ask a question in a Q&A session in Slido

Slido is available in Webex Meetings and Webex Webinars. You can use Slido in Webex Meetings on [version 41.6 and later sites](https://help.webex.com/qgij81/), and in Webex Webinars on [version 41.9 and later sites](https://help.webex.com/qgij81/) and in Webex Suite meeting platform. Slido polls and Q&A in webinars are available for up to 10,000 attendees, while quizzes are available for up to 5,000 participants.

| 1 | In the Q&A panel, type your question.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | (Optional) Select Add label and choose the label that indicates which topic your questions relates to. Labels are available if the meeting host has enabled them for participants.                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| 3 | Choose whether to send your question anonymously or using your name: To use your name, select ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/461001-462000/461863.jpg) and choose your name.To send your question anonymously, select ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/461001-462000/461863.jpg) and choose Anonymous.The host, cohost, and other participants won't be able to identify you if you send your question anonymously.If you don't see the Anonymous option, the host has disabled anonymous questions. |
| 4 | Select Send. If the host is moderating questions, the questions are reviewed first before they go live for everyone to see. If you need to edit or withdraw the question, select ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/458001-459000/458469.jpg). You have five minutes to edit or withdraw the question after you send it.                                                                                                                                                                                                                                               |

If you see the message, Questions closed in the Q&A panel, the host has closed the Q&A session to any more questions.
