# View reports for your Webex site

## Reports overview

You can generate and view the following types of reports:

- Common (for the entire site)
- Webex Events
- Webex Support
- Webex Training
- Remote Access
- Access Anywhere
- WebACD

Each report displays the time stamp using the Greenwich Mean Time (GMT) time zone. 

Many reports can be run in 15, 30, and 60-minute intervals.

Usage reports that show IP addresses in WBS39.4 and later will include mobile IP addresses.

## View usage reports

View data about how many participants attended a session and how many minutes the session lasted. 

| 1 | Sign in to Webex Site Administration and go to  Reports  \>  Common  \>  Usage Report .                                                                                                                                                                                                                                                                                |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | From the Service  drop-down list, choose a service or All .                                                                                                                                                                                                                                                                                                            |
| 3 | (Optional) To run a report about the meetings that a specific user hosted, in the User Name  field, enter the user name.                                                                                                                                                                                                                                               |
| 4 | Select the beginning and ending dates for the report. Report data is only available for sessions conducted up to three months before the current date.                                                                                                                                                                                                                 |
| 5 | (Optional) To view a report for only certain session topics, in the Topic  box, enter all or part of the topic name. You can use wildcards, such as ?  or \* , when specifying a text string.                                                                                                                                                                          |
| 6 | To sort the report by date, topic, username, or start time, from the Sort results by  drop-down list, select the sort criteria.                                                                                                                                                                                                                                        |
| 7 | Click Display Report . For more information about how to read your usage report, see [ Usage Report Terms and Definitions ](https://help.webex.com/ng1kl5s).                                                                                                                                                                                                           |
| 8 | To export the report in CSV format so that you can import it into a spreadsheet or other program, at Export CSV Reports , do one or both of the following: To export the Summary Usage report to a CSV file, select Session Summary . To export the Attendee Detail report for all sessions on the Summary Usage Report  page to a CSV file, select Attendee Details . |

## View recording storage usage report

View how much storage was used for recordings made in the last 90 days.

| 1 | Sign in to Webex Site Administration and go to Webex Site Administration and go to Reports \> Common \> Recording Usage.                                                                                                                                                                                                                                           |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Select the beginning and ending dates for the report.                                                                                                                                                                                                                                                                                                              |
| 3 | (Optional) To view a report for a specific recording, in the Recording Name field, enter all or part of the topic name. You can use wildcards, such as ? or \*, when specifying a text string.                                                                                                                                                                     |
| 4 | (Optional) To run a report about the meetings that a specific user hosted, at Host user name enter the host user name. Report data is only available for sessions conducted up to three months before the current date.                                                                                                                                            |
| 5 | To sort the report by date, topic, user name, or start time, from the Sort results by drop-down list, select the sort criteria.                                                                                                                                                                                                                                    |
| 6 | Select Display Report.                                                                                                                                                                                                                                                                                                                                             |
| 7 | To export the report in CSV format so that you can import it into a spreadsheet or other program, at Export CSV Reports, do one or both of the following: To export the Summary Usage report to a CSV file, select Session Summary. To export the Attendee Detail report for all sessions on the Summary Usage Report page to a CSV file, select Attendee Details. |
