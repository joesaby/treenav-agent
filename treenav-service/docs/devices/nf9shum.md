# Enable Activity Indicators on Board, Desk, and Room series Devices

Activity indicators are enabled by default. You can configure them from the local web interface of the device. Read the [Device Configurations ](https://help.webex.com/en-us/n5pqqcm/Advanced-Configurations-for-Room-and-Desk-Devices-and-Webex-Boards) article for more information on how to access it.

When the web interface of the device is opened, navigate to the Setup  tab, select Settings , and choose the following configuration:

- Configuration  \> UserInterface Whiteboard  \> ActivityIndicators : on or off.
