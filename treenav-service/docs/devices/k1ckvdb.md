# Set the phone wallpaper on Cisco Multiplatform Phones

You can select the wallpaper (background) for the phone screen.

| 1 | Press Applications ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381337.jpg). |
| - | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select User preferences \> Screen preferences \> Wallpaper.                                                                                  |
| 3 | Press Select to scroll through the list and select a wallpaper.                                                                              |
| 4 | Press Set to apply the selection.                                                                                                            |
