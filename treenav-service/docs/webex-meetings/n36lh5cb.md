# Video Device-Enabled Webex Meetings Recordings

The standard format for video device-enabled meeting recordings is MP4.

When you receive your recording after the meeting, you can:

- Share the recording.
- Download the recording, which includes the sharing, audio, and video portions of the meeting.
- Stream MP4 recordings, including video, using our web-based player that supports Flash and HTML 5.
- Set a password for the recording.
- Edit the topic and description of the recording file.
- Download and play MP4 files on almost any device and edit them using standard MP4 editors.

You cannot control which Webex panels appear, set partial playback, or choose to include or exclude network recording player controls.
