# Webex App | Upload Files to Google Drive

| 1 | In the message area, click Attachment ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/422001-423000/422517.jpg), and then select Google Drive. |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Go to the Upload tab in Google Drive.                                                                                                                                                         |
| 3 | Select the file from your desktop, then click Upload.                                                                                                                                         |
