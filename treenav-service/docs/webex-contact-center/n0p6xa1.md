# Data Locality in Webex Contact Center

When you're setting up a Webex Contact Center, there's a drop-down list called Country of Operation. This list shows the countries where Webex Contact Center is available. If you want to see which countries are included, you can look at the _Geographic Terms and Restrictions_ section in the [Offer Description: Cisco Collaboration Flex Plan Contact Center](https://www.cisco.com/c/dam/en%5Fus/about/doing%5Fbusiness/legal/OfferDescriptions/cisco%5Fcollaboration%5Fflex%5Fplan%5Fcontact%5Fcenter.pdf)article.

When you pick a Country of Operation in the set up wizard under the Contact Center tab, the system will create your tenant, in the data center that's closest to that country. The tables below show which data centers are used for each country you can choose.

The Webex Contact Center tenants host data related to: 

- Tenant Configuration
- Call Data Records
- Call Recordings
- Reporting
- Digital Channels

For the list of data centers to view where Chat data resides, see the [Data Residency in Webex](https://help.webex.com/en-US/article/oybc4fb/Data-Residency-in-Webex) article.
