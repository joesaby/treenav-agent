# Edit a virtual agent in Webex Contact Center

| 1 | Log in to the customer organization at <https://admin.webex.com>.                                                                  |
| - | ---------------------------------------------------------------------------------------------------------------------------------- |
| 2 | From the navigation pane on the left, in the Servicessection, click Contact Center \> Tenant Settings \> Integrations \> Features. |
| 3 | Select the  Virtual Agent  tab and choose the virtual assistant that you want to edit.                                             |
| 4 | Modify the attribute that you want to change.                                                                                      |
| 5 | Click Finish  to save your edits.                                                                                                  |
