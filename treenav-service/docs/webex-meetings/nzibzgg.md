# SIP TLS 1.2 Requirement for Webex App

The Payment Card Industry (PCI) Security Standards Council recommends that organizations stop using Secure Socket Layer (SSL) and Transport Layer Security (TLS) 1.0, and upgrade to TLS 1.2 as soon as possible. See [Migrating from SSL and Early TLS ](https://www.pcisecuritystandards.org/pdfs/PCI%5FSSC%5FMigrating%5Ffrom%5FSSL%5Fand%5FEarly%5FTLS%5FResource%5FGuide.pdf).
