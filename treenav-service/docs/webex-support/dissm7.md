# Change Access Code or Phone Authentication Options for Remote Access Groups

| 1 | In Site Administration, go to Configuration \> Remote Access \> Manage Groups.                                                             |
| - | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | In Site Administration, go to Configuration \> Remote Access \> Groups.                                                                    |
| 3 | Locate the computer for which you want to change options.                                                                                  |
| 4 | In the Actions column, select the icon for the group.                                                                                      |
| 5 | In the Access code or Phone authentication section, specify the access code or phone authentication options.                               |
| 6 | If you want to apply the same access code settings to all lower-level groups and computers within the group, select Apply to Lower Levels. |
| 7 | If you want to prevent those access code settings from being changed, select Disallow changes for all lower level groups and computers.    |
| 8 | Select Update to save your settings.                                                                                                       |
