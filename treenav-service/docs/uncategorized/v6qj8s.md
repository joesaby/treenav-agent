# Find device information

Here is some of the information that can display:

- Phones and Analog Telephone Adapters (ATA):

  - Status
  - IP address
  - MAC address
  - Time zone
  - Country
  - Emergency service address
  - T.38 Fax Protocol ON/OFF (ATA only. Not all PSTN service providers support this protocol.)
  - Input Audio Level (ATA only)
  - Output Audio Level (ATA only)
- Webex Room Devices:

  - Status
  - IP address
  - MAC address
  - Software version
  - Serial number
  - Potential issues
  - Reported issues

| 1 | Sign in to Control Hub at <https://admin.webex.com> and open the Devices page. |
| - | ------------------------------------------------------------------------------ |
| 2 | Select a device from the list and review the data about the device.            |
