# Supported browsers for Webex Share

Recommended browsers are the latest supported versions of:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Apple Safari

Some browsers require you to grant permission to share your computer screen before Webex Share can function correctly. Select the site icon in your address bar and give permission to share your screen.
