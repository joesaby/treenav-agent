# Set Up a Customer Organization in Control Hub

Before you begin

You can't register on-premises resources for Hybrid Services, the customer administrator must perform these steps.

| 1 | From the partner view in <https://admin.webex.com>, go to Customers, select the customer that you want to set up. |
| - | ----------------------------------------------------------------------------------------------------------------- |
| 2 | Click Set up Customer or View Customer to launch the customer's management portal and provide setup assistance.   |
