# Trim the beginning and end of your Webex Meetings, Webex Webinars, or Webex Events (classic) recording

This feature is available for [video-centric network-based MP4 recordings](https://help.webex.com/d8o7deb/).

The feature isn't available for data centers in China or Video Platform version 1.0 sites. FedRAMP-authorized Webex sites must upgrade to Webex for Government to get the feature with Video Platform version 2.0\. For browser requirements, see [Browser support for video-centric network-based MP4 recordings](https://help.webex.com/nki3xrq/Webex-Meetings-Suite-System-Requirements#id%5F116436).

If you shared a video file using Share File during the meeting, webinar, or event, the MP4 recording won't be saved in video-centric format. 

The trimmed recording is only available when played back from the Webex site or the recording link. When downloaded and played back, the full recording is available.

| 1 | [Sign in to your Webex site](http://signin.webex.com) and go to Recordings.                                                                                                                                                                                                                                                                                                                                                                                                                                |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Locate the recording that you want to edit and select its name.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 3 | Click Play ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/446001-447000/446491.jpg) and select Edit playback range ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/435001-436000/435655.jpg) in the lower right of the screen.                                                                                                                                                                              |
| 4 | To trim the beginning of the recording, select Preview playback range ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/446001-447000/446491.jpg) to start playing the recording. When you determine where you want the recording to start, select Pause![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/439001-440000/439110.jpg) and drag the **playback start control** there.                               |
| 5 | To trim the end of the recording, select Preview playback range ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/446001-447000/446491.jpg) to start playing the recording. Drag the seekbar to determine where you want the recording to end. When you find the end point, select Pause![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/439001-440000/439110.jpg) and drag the **playback end control** there. |
| 6 | Select Save playback range ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/446001-447000/446490.jpg) and exit the meeting window.                                                                                                                                                                                                                                                                                                                           |
| 7 | From Recordings, click and open the recording again to make sure it starts and ends where you expect. If you want to revert the changes that you made, select Edit playback range ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/435001-436000/435655.jpg), then Restore original playback range ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/435001-436000/435691.jpg).                                 |
