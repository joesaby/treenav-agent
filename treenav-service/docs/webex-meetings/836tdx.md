# Webex Secure HTML Tags and Attributes

Your Webex site allows you to use HTML code to customize branding. For example, when customizing your site such as the left navigation bar, or email templates. Unsecured HTML tags and attributes, and JavaScript code, are not supported. 

HTML code only supports the following secure HTML tags, attributes, and CSS properties. Only valid URLs that use the supported protocols listed in the following table can be entered in URL fields.
