# Serviceability messages for Board, Desk, and Room series

You can find serviceability messages in the Issues and diagnostics  page. On the home screen, swipe right and select Device settings . If the device has detected an issue, a message appears in the issues and diagnostics page.

- Network ports blocked : This may impact the call quality. This issue may be caused by a firewall blocking media on UDP port 33434.
- Network Link Status : Make sure that the Ethernet cable is connected or that the Wi-Fi is configured correctly.
- High temperature : The system temperature is too high. This may impact the system performance. Try to lower the room temperature and block any direct sunlight to the system.
- Main display not detected : Please check that the main display is turned on and connected to the correct display output on the video system. See the article on monitors for tips on how you can fix this issue. Read more on this issue in the [ External monitor settings for Room Series devices](https://help.webex.com/nyi4lcq/) article.
- Automatic updates disabled : The system software will not be updated automatically.
