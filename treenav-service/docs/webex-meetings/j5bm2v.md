# Configure Webex Scheduler with Google Calendar in Webex Site Administration

Site administrators who want to install the [Cisco Webex for G Suite](https://workspace.google.com/marketplace/app/cisco%5Fwebex/678783785621) add-in for all users in their organization can sign in to their G Suite administrator accounts and follow the standard Google process to install G Suite add-ins through a group policy or through default preferences at [https:/​/​support.google.com/​chrome/​a/​answer/​188453?hl=en ](https://support.google.com/chrome/a/answer/188453?hl=en).

| 1 | Sign in to Webex Site Administration and go to Configuration  \>  Common Site Settings \>  Options.                                                                                                                                                                                                                                       |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the  Third-Party Integration  section, under  Google, check  Calendar. You can select which Google domains can utilize this feature in the  Restrict the integration to users in these G Suite domains field. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/430001-431000/430531.jpg) |
| 3 | Click Update.                                                                                                                                                                                                                                                                                                                             |
