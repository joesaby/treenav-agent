# Enable Webex Productivity Tools in Microsoft Outlook for Windows

If you’ve installed Webex Productivity Tools, but you're not seeing Webex options within Outlook, you can enable Productivity Tools. 

| 1 | Check your Webex Productivity Tools preferences: From the Windows Start menu, select **Webex Productivity Tools** Select the **Settings** icon. Choose **Preferences** from the menu. Select **Tools**. Make sure that the **Microsoft Outlook** option is checked.                          |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Check your Microsoft Outlook options: In Microsoft Outlook, select **File > Options > Add-Ins**. Next to "Manage," select **Disabled Items**, and then select **Go**. In the Disabled Items dialog box, select **Add-in: WebexOIAddin(ptolkadd.dll)**. Select **Enable** and then **Close**. |
