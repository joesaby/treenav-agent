# Cancel a Cisco Webex Services Trial Subscription in Control Hub

| 1 | From the customer view in <https://admin.webex.com>, click Account under Management in the left navigation menu.                                                                                              |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the Subscriptions tab.                                                                                                                                                                                 |
| 3 | Click on the trash ![Trash icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502697.jpg) icon next to the trial subscription you want to cancel. |
| 4 | Click Yes.                                                                                                                                                                                                    |
