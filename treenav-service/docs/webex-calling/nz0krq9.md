# Configure and manage your Webex Calling users

Determine the method to add users based on the number of users you wish to add in Control Hub, whether you manually add each user by email address or add multiple users using a CSV file. The choice is yours.

- [Modify users with the CSV template](https://help.webex.com/ih07j)
- [Edit service licenses for individual users](https://help.webex.com/9dmqgv)
- [Add or delete numbers for a user](https://help.webex.com/n4kem8b)
- [Resend an invitation email](https://help.webex.com/l08e04)
- [Delete a user from your organization](https://help.webex.com/0qse04/)
