# Set recording settings in Webex Meetings

Recording settings only apply to cloud-based recordings. 

| 1 | Sign in to [User Hub](https://globalpage-prod.webex.com/signin), then select Settings.                                                                          |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the Recording tab. ![Recording tab](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/440001-441000/440786.jpg) |
| 3 | For Recording transcripts, select [create recording transcripts for your MP4 recordings](https://help.webex.com/nk62t5eb/).                                     |
| 4 | For AI summary and Chapters, select [generate AI summaries and chapters for your MP4 recordings](https://help.webex.com/article/nke3ax3/).                      |
| 5 | For Recording layout, [select the screen layout for recordings](https://help.webex.com/en-us/k2o4e3/).                                                          |
| 6 | Select Save.                                                                                                                                                    |
