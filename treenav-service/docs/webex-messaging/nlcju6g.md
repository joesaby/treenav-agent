# Data classifications for Meetings and Messaging Spaces

Space Classification is part of Pro Pack for Control Hub. For more information, see [Pro Pack For Cisco Webex Control Hub](https://help.webex.com/np3c1rm/).

Space classifications are labels that remind users what they can and can't talk about or share in a space. When a space is classified, a classification label—for example, **Public** or **Company Sensitive**—appears in the message area.

You can add, arrange, and edit space classifications to match your organization's information policies. Users decide which classification to use when they create new spaces.

Webex now also supports classifications for meetings : 

- [Webex App | Schedule a meeting from the meetings calendar](https://help.webex.com/article/wy517z/)
- [Webex App | Schedule a meeting from a space](https://help.webex.com/article/nu5w9x3/)
- [Schedule Webex Meetings from a mobile device](https://help.webex.com/article/qxzmuf/)
- [Schedule a meeting from User Hub](https://help.webex.com/article/xm3o0v/)
- [Schedule and join meetings with Webex Scheduler for Microsoft Outlook](https://help.webex.com/article/nttajz6/)

 Use the same set of classifications for both meetings and messaging. You can now find messaging space classification settings in Control Hub. 

## Set up classified spaces

| 1 | Sign in to [Control Hub](https://admin.webex.com/) then go to Management \> Organization Settings.                                                                                     |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Navigate to Classified meetings and messaging spaces and turn on Allow classifications.                                                                                                |
| 3 | For each classification you want to create, click Add a classification, enter a name and description, and then click Next.                                                             |
| 4 | (Optional) Select and change the default classification for messaging spaces.                                                                                                          |
| 5 | Drag and drop the classification names to arrange them in order from the most restricted classification (top of the list) to the least restricted classification (bottom of the list). |
| 6 | Click Save.                                                                                                                                                                            |

## Make changes to classifications

| 1 | Sign in to [Control Hub](https://admin.webex.com/) then go to Management \> Organization Settings.                                                                                          |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Navigate to Classified meetings and messaging spaces and click Configure.                                                                                                                   |
| 3 | Select a classification name, and then choose from the following: Rename—Edit the classification name and description. Disable—Removes the classification from the list available to users. |
| 4 | (Optional) Select and change the default classification for messaging spaces.                                                                                                               |
| 5 | To save you changes, click Done.                                                                                                                                                            |
