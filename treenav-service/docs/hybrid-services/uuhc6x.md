# License Requirements for Hybrid Services

- [Calendar Service](#reference%5F623B0BBD21A97B4F8277747CB49FA098)
- [Data Security](#reference%5FDE3FB4DE51E90628770122E31F247417)
- [Video Mesh](#reference%5F5B9678E9947BA70D9AE3A4C73821B8D4)
- [Message Service](#id%5F124398)

**Webex License Requirements for Hybrid Calendar** 

- A Webex organization with a paid subscription.

**Other Cisco Licenses**

- [Cisco Expressway](https://software.cisco.com/download/navigator.html?mdfid=285978528&catid=278875240) (free software connector license with hybrid services)

**Documentation**

<https://www.cisco.com/go/hybrid-services-calendar>

**Webex License Requirements for Hybrid Data Security** 

- A Webex organization with a paid subscription
- Pro Pack for Cisco Webex Control Hub (See <https://www.cisco.com/go/pro-pack>)

**Other Cisco Licenses**

- N/A

**Documentation**

<https://www.cisco.com/go/hybrid-data-security>

**Webex License Requirements for Video Mesh** 

- A Webex organization with a paid subscription that includes Business Messaging and Advanced Meetings at a minimum.

**Other Cisco Licenses**

- Webex video platform 2.0 (You can verify that your Webex site is on video platform version 2.0 if it has the Media Resource Type list available in the Cloud Collaboration Meeting Room site options.)

**Documentation**

<https://www.cisco.com/go/video-mesh>

**Webex License Requirements for Hybrid Message**

- You need any paid-for Webex app offer for your organization. You can order this through Cisco Commerce Workspace.  
You also need to have access to Cisco Webex Control Hub, with administrator privileges for your organization (you get these as part of the ordering process).
- You should import all Jabber users into Control Hub and grant them all the "Message Free" entitlement. This entitles all the Jabber users to the basic Webex App messaging functionality.  
There are no additional paid license requirements for this basic messaging. Having this entitlement for all users improves interoperability between those who are enabled for Hybrid Message and those who are only licensed for Jabber.

**Other Cisco Licenses**

- We assume that all users are previously licensed for Cisco Jabber, with Jabber registered to Cisco Unified Communications Manager IM and Presence.

**Documentation**

<https://www.cisco.com/go/hybrid-services-message>
