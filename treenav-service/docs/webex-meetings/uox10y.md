# Enable Moderated Unmute Mode for Webex Meetings

This feature is available for 41.2 sites and later.

The following application versions support Moderated Unmute mode:

- Mac and Windows—41.1 and later

- [Site Administration](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5F7ea1ec1b-3104-4634-8d15-7a5aaa139432)
- [Control Hub](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5Fe5de5a81-316d-4bd7-b944-eb75dea8be9a)

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \>  Options.                                               |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Scroll down to Site Options and check the box next to Enable "Moderated unmute mode," which allows the host and cohosts to unmute participants. |
| 3 | (Optional) Click Turn on "Moderated unmute mode" by default to make Moderated unmute mode the default for your site.                            |

| 1 | Sign in to [Control Hub](https://admin.webex.com).                                                                         |
| - | -------------------------------------------------------------------------------------------------------------------------- |
| 2 | Go to Services \> Meeting \> Sites.                                                                                        |
| 3 | Select your Webex site.                                                                                                    |
| 4 | Under Common Settings, select Security.                                                                                    |
| 5 | In the Other section, toggle Enable ‘Moderated unmute mode’, which allows the host and co-hosts to unmute participants on. |
| 6 | (Optional) Toggle Turn on ‘Moderated unmute mode’ by default on to make Moderated unmute mode the default for your site.   |
