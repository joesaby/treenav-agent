# Configure meeting options for your site

## Allow users to add session information to calendars

This procedure is for Webex Meetings, Webex Training, and Webex Events only. Use this procedure to enable iCalendar functionality on your site, and to show the Add to My Calendar button in the Webex session information page. iCalendar is a standard calendar format. With iCalendar, users can share calendar information and automatically update their schedules. 

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                  |
| - | ----------------------------------------------------------------------------------------------------------------- |
| 2 | In the Site Options section, check the iCalendar (Show Add to My Calendar for Meetings, Training and Events) box. |
| 3 | Select Update.                                                                                                    |

## Allow Cisco Unified MeetingPlace users to join external meetings WBS30

This procedure is for Meeting Center and WBS30 only. Set the default behavior in the meeting scheduler and Productivity Tools scheduler for whether to allow attendees located on the public Internet to join a meeting. The option first must be turned on for a meeting host in order to allow external attendees to join a meeting. If the setting is not selected in the scheduler, only attendees within the same internal network can join a meeting. 

Before you begin

 This option is available only if your Webex site is provisioned with the following: 

- The Cisco Unified MeetingPlace integration option
- The Webex Node for MCS option

| 1 | Sign in to Webex Site Administration and go to Users \> Add User. To update an existing user account, select Users \> Edit User, then select a user. |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Privileges section, next to Meeting Center, select Allow external attendees.                                                                  |
| 3 | Select Add or Update to save your changes.                                                                                                           |
| 4 | On the left navigation bar, select Configuration \> Common Site Settings.                                                                            |
| 5 | Scroll down to Cisco Unified MeetingPlace Integration Settings.                                                                                      |
| 6 | To allow external attendees, check the Default MP external attendees box.                                                                            |
| 7 | Select Update.                                                                                                                                       |

## Enable text messaging

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options. |
| - | ------------------------------------------------------------------------------------------------ |
| 2 | Under Site Options, check the Enable text messaging (SMS) box.                                   |
| 3 | Select Update.                                                                                   |

## Allow users to schedule meetings and training sessions for other users

This procedure is for Webex Meetings and Webex Training only.

Before you begin

Users who schedule meetings must have a Webex host account.

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                    |
| - | ------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Schedule Meeting Options section, check the Users can let other users schedule meetings on their behalf box. |
| 3 | Select Update.                                                                                                      |

## Allow users to print or save the file being shared

This procedure is for Webex Meetings, Webex Events, and Webex Training only. 

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                                                                           |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Site Options section, check the  Allow Print/Save in Document Sharing box. Uncheck the box to prevent attendees from printing or downloading a file during sharing. |
| 3 | Select Update.                                                                                                                                                             |
