# Webex App | Show or hide names in thumbnail videos

When names are hidden, you can still see the name of an individual participant by hovering over that person's video.

The active speaker’s name always shows so you know who's talking.

- [Desktop](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5Fb93be2d6-eb7b-482a-982e-1390fc3bfd77)
- [Mobile](#task-template%5Fed7eabef-08ac-4e90-a786-f40147d29188)

| 1 | When you're in a meeting, click Layout ![Layout](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/455001-456000/455202.jpg). When you're not in a meeting, click Settings ![Settings button represented by a cog icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/427001-428000/427579.jpg) on the [navigation sidebar](https://help.webex.com/nx3i8h5/) and select Meetings \> Layout. |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Toggle off or uncheck Automatically hide names to show the names in the videos. If you decide that you want to hide the names, toggle on or check Automatically hide names.                                                                                                                                                                                                                                                                                                       |

We save the value that you select in Settings  so that it applies to all future meetings.

| 1 | When you're in a meeting, tap Layout ![Layout](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/455001-456000/455202.jpg).                                              |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Toggle off Automatically hide names to show the names in the videos. If you decide that you want to hide the names, toggle on Automatically hide names. We save the setting so it applies to all future meetings. |
