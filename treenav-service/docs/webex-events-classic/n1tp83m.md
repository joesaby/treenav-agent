# Manage Scheduling Templates for Webex Events in Cisco Webex Site Administration

## Scheduling Templates for Webex Webinars

A scheduling template defines many of the settings for a scheduled event. If hosts often use the same settings for their events, a site administrator can save those settings in a template. When scheduling events, a host can select the template to avoid having to set many of the same options for each event. 

There are two types of scheduling templates: 

- Standard scheduling templates: Available for all hosts. A Webex service includes a set of standard templates, and site administrators can also make hosts’ personal templates available as standard templates.
- Personal scheduling templates: A template that a host sets up and that is available only for that host. A host can create a personal template from within the event scheduler, by using the Save As Template option. When saving a template, the host can also save the template as a standard template, by selecting Save As Standard Template. The template then appears in the list of standard templates.  
After saving the template, only the host images that were uploaded remain. All other files uploaded to the template like the meeting materials and the image description are deleted.  
In the list of standard templates for Webex Webinars, site administrators can:
- Show or hide any standard template, including those templates that hosts create and save as a standard template.
- Make any standard template the default template for Webex Webinars.
- Delete a standard template from a Webex site.

## Show or Hide Scheduling Templates for Webex Events

| 1 | Sign in to Webex Site Administration and go to Configuration \> Webex Events \>  Options.                                  |
| - | -------------------------------------------------------------------------------------------------------------------------- |
| 2 | Scroll down to Standard Scheduling Templates.                                                                              |
| 3 | In the Make Visible column, do one of the following: Check the box to show a template. Uncheck the box to hide a template. |
| 4 | Click Update.                                                                                                              |

## Set the Default Scheduling Template for Webex Events

| 1 | Sign in to Webex Site Administration and go to Configuration \> Webex Events \>  Options.     |
| - | --------------------------------------------------------------------------------------------- |
| 2 | Scroll down to Standard Scheduling Templates.                                                 |
| 3 | In the Set as Default column, select the template that you want to make the default template. |
| 4 | Click Update.                                                                                 |

## Delete Scheduling Templates for Webex Events

| 1 | Sign in to Webex Site Administration and go to Configuration \> Webex Events \>  Options. |
| - | ----------------------------------------------------------------------------------------- |
| 2 | Scroll down to Standard Scheduling Templates.                                             |
| 3 | Check the corresponding check boxes for the templates that you want to delete.            |
| 4 | Click Delete Template(s).                                                                 |
