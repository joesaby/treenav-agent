# Stream Webex Meetings and Events to Facebook Live

When live streaming is enabled, it's enabled in both Webex Meetings and Webex Events (classic).

- [Site Administration](#id%5F133979)
- [Control Hub](#id%5F133978)

Enable streaming to Facebook Live on your Webex site

Users can stream to any streaming provider you enable in Site Administration. Before your users can stream to Facebook Live, you must enable the Facebook Live setting and the live streaming setting for each user.

| 1 | Sign in to Webex Site Administration. On the left navigation bar, go to Configuration \> Common Site Settings \> Options. |
| - | ------------------------------------------------------------------------------------------------------------------------- |
| 2 | Scroll to the Live Streaming Services section and select Facebook Live.                                                   |
| 3 | Click Update.                                                                                                             |

Enable the live streaming setting for each user

| 1 | Sign in to Webex Site Administration. On the left navigation bar, go to User Management \> Edit User and select a user account to update. |
| - | ----------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Privileges section for that user, check the Enable Live Streaming box.                                                             |
| 3 | Click Update.                                                                                                                             |

Enable streaming to Facebook Live on your Webex site

Users can stream to any streaming provider you enable in Control Hub. Before your users can stream to Facebook Live, follow the steps below to enable the Facebook Live setting and the live streaming setting for each user.

| 1 | Sign in to [Control Hub](https://admin.webex.com/), then under Services, select Meeting. |
| - | ---------------------------------------------------------------------------------------- |
| 2 | On the Sites tab, select a Webex site, then click Settings.                              |
| 3 | Select Common Settings \> Site Options.                                                  |
| 4 | Scroll to the Live Streaming section and turn on Facebook Live.                          |
| 5 | Click Save.                                                                              |

Enable the live streaming setting for each user

| 1 | Sign in to [Control Hub](https://admin.webex.com/), go to Management \> Users.                              |
| - | ----------------------------------------------------------------------------------------------------------- |
| 2 | Select a user, then select Meetings.                                                                        |
| 3 | From the Settings apply to drop-down menu, select the meeting site to update, then click Advanced Settings. |
| 4 | In the User Settings section for that user, check Enable Live Streaming / NDI.                              |
| 5 | Click Update.                                                                                               |
