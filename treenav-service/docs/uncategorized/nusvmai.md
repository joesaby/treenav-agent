# Start a Meeting in the Cisco Webex Meetings Virtual Desktop App

## Start a Meeting in Your Personal Room

| 1 | Open the virtual desktop app and sign in.                                                                                                                                                                                                                                                                                                   |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Click the Start Meeting ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502391.jpg) button. The Start Meeting can only start a Personal Room meeting. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502395.jpg) |

## Start a Scheduled Meeting

| 1 | Open the virtual desktop app and sign in.                                                                                                                                                                                                                                                                               |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the Start ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502347.jpg) button to start the meeting that you're the host of. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502394.jpg) |
