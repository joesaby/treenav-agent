# Manually Install the Webex Content Sharing Extension for Google Chrome

| 1 | [Download the Cisco Webex Content Sharing extension](https://www.cisco.com/c/dam/en/us/td/docs/collaboration/webex%5Fcenters/esp/Cisco%5FWebex%5FContent%5FSharing%5Fmain%5Fcrx.zip) for Google Chrome.                                            |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Save the file Cisco\_Webex\_Content\_Sharing\_main\_crx.zip file to your desktop or other convenient location.                                                                                                                                     |
| 3 | Unzip the file. The unzipped folder contains the Cisco\_Webex\_Content\_Sharing\_main.crx extension file.                                                                                                                                          |
| 4 | Open a new tab in Chrome, type chrome://extensions in the Address Bar, and then press Enter.                                                                                                                                                       |
| 5 | In the Chrome Extensions tab, turn on Developer mode in the top right-hand corner of the page, and then refresh the page. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/430001-431000/430195.jpg) |
| 6 | Drag the Cisco\_Webex\_Content\_Sharing\_main.crx extension file into the Chrome Extensions tab.                                                                                                                                                   |
| 7 | Click Add extension.                                                                                                                                                                                                                               |
