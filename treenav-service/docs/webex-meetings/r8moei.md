# Assign a certificate to a Webex session type

| 1 | Sign in to Site Administration and go to Configuration \> Common Site Settings \> Session Types. |
| - | ------------------------------------------------------------------------------------------------ |
| 2 | Select the link for the desired session type.                                                    |
| 3 | Scroll down to PKI encryption and check the box next to it.                                      |
| 4 | Select Update.                                                                                   |
