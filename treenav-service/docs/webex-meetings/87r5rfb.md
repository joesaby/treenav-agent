# Rename a meeting, webinar, event, or training session recording

| 1 | Sign in to [User Hub](https://globalpage-prod.webex.com/signin).                                                                                                                       |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Recordings from the navigation panel. On the Recordings tab, you can see recordings for all the meetings, webinars, events, and training sessions that you hosted and recorded. |
| 3 | Click Edit next to the name of the recording.                                                                                                                                          |
| 4 | Edit the name and description, and choose which panels you want to include in the recording.                                                                                           |
| 5 | Clear the Transcripts check box if a [transcript was created](https://help.webex.com/nk62t5eb/) but you don't want others to see it when they play the recording.                      |
| 6 | Select Save.                                                                                                                                                                           |
