# Find information about your desk or conference phone

| 1 | Do one of the following actions: Desk phones—Press Applications ![the Applications button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381337.jpg). Conference phone—Press Settings.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Status \> Product Information.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 3 | View information about your phone Product name—Model number of the Cisco IP Phone. Serial number—Serial number of the Cisco IP Phone. MAC address—Hardware address of the Cisco IP Phone. Software version—Version number of the Cisco IP Phone firmware. Hardware version—Version number of the Cisco IP Phone hardware. Certificate—Status of the client certificate, which authenticates the Cisco IP Phone for use in the ITSP network. This field indicates if the client certificate is properly installed in the phone. Customization—For an RC unit, this field indicates whether the unit has been customized or not. Pending indicates a new RC unit that is ready for provisioning. If the unit has already retrieved its customized profile, this field displays the Customization state as Acquired. |
| 4 | Do one of the following actions: 7800, 7832, 6800, 8832 —Press Back. 8800—Press Exit.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
