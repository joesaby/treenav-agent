# Webex App | Turn off your video for all incoming calls

- [Windows](#id%5F125416)
- [Mac](#id%5F125418)
- [Mobile](#id%5F133513)

| 1 | Click your [ profile picture](https://help.webex.com/en-us/m67xht/Open-Settings-from-Your-Profile-Picture-in-Cisco-Webex-Teams), select Settings, and then go to Calling.                                                                                                           |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Uncheck the Answer calls with my video on check box and then click Save. Webex App supports [ different types of calls](https://help.webex.com/z94f4f/Calling-Behavior-in-Cisco-Webex-Teams). If you don't see this check box, you're not set up with the required calling options. |
