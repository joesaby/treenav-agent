# Webex App | Microsoft OneDrive and SharePoint Online

## Add your account

Get access to your company's content management system without leaving Webex App. Share and edit files in a space with another person, or with your team.

- Enterprise accounts––Your administrator gives you permissions to access your company's enterprise content management system.
- Service provider accounts––Contact your provider to get access to your company's content management system.

Already have permissions? [Add your Microsoft OneDrive or SharePoint Online account](https://help.webex.com/en-us/article/nymj5jk/) to your desktop or mobile app.

## Upload a file or folder

[Upload files to Microsoft OneDrive or SharePoint Online](https://help.webex.com/article/n9vrrceb) on the desktop app, so you're ready to share them in a space. Limit access to view-only if you're sharing informational content, allow editing permissions for team collaboration and discussions. 

## Share a file

[Share files from Microsoft OneDrive or SharePoint Online](https://help.webex.com/article/n9p506) on the desktop or mobile app and start collaborating with your team.

## Reshare a file

[Reshare files from Microsoft OneDrive or SharePoint Online](https://help.webex.com/article/njcmj) on the desktop or mobile app to give new users access to previously shared files. 

## View and edit a file

[View and edit Microsoft OneDrive or SharePoint Online files](https://help.webex.com/en-us/article/hhhpd4/) depending on the shared permissions. The file opens in the application that's associated with the file type on your desktop or mobile.

## Link a folder to a space

[Link a Microsoft OneDrive or SharePoint Online folder to a space](https://help.webex.com/article/n4ve41eb) to upload multiple files, organized and easy to find in the space's Content tab.

## View or edit a file in a linked folder

[View or edit files in a linked Microsoft OneDrive or SharePoint Online folder](https://help.webex.com/article/pp213c) in the Content tab of the desktop or mobile app.
