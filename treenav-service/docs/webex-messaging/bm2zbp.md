# Webex App | View or download files shared in a space

When someone shares multiple images in a message, you'll see them in a unified gallery view within the message on both desktop, and mobile devices.

- [Desktop](#task%5F7AC1A30967E643F3C183F36C3E3AE330)
- [iOS](#task%5F0050F2B60BB341A767FB294F5DB85187)
- [Android](#task%5F7FE754D2DE8A209C04DCBBAB34CD965A)
- [Web](#Cisco%5FTask.dita%5F8e581ad8-91b7-4258-bedb-8e2696975fcf)

| Go to Messaging ![Messaging button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/438001-439000/438646.jpg), choose a space, click Content, and then select Files. Select from List view![List view](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/454001-455000/454478.jpg), or Grid view![Grid view](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/454001-455000/454479.jpg). You can also click on a file in a space to see the file in a preview window, then click left ![Navigate left icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/456001-457000/456776.jpg) or right ![Navigate right icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/456001-457000/456777.jpg) to quickly navigate between other files shared in the space. When you hover over a file, or open a file in a preview window, you can do the following: ![Show in conversation](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/440001-441000/440620.jpg) Show in conversation. ![Download icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502593.jpg) Download, and then click Show in folder ![Show in folder icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/500001-600000/500001-510000/502001-503000/502594.jpg). You can [choose where to save downloaded files](https://help.webex.com/n63irlr/). ![Update file share icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/454001-455000/454477.jpg) Update file share—If the file is shared from Microsoft OneDrive or SharePoint Online. Your organization administrator can choose to block either file downloads, or file previews. That means you could be limited to just previewing files, or just downloading them, depending on what your administrator has set up. |

| Choose how to view and download files: Preview—Tap the file. Tap ![More Options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/453001-454000/453339.jpg) and then choose from: Share—choose an application to share the file. Save—save the file to the device. Open with...—choose an application to open the file. Show in Space—see who sent a file and when they sent it. See all files—In the space, tap the activity menu ![More Options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/453001-454000/453339.jpg), then tap Content and choose Files. Tap a file to see a preview, then swipe left or right to quickly navigate between files shared in the space. That means you could be limited to just previewing files, or just downloading them, depending on what your administrator has set up. |
