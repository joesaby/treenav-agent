# Webex App | Send a message

This describes how to send a message in an existing space. To start a conversation in a new space, see [Webex App | Start a conversation](https://help.webex.com/zi4dpc/).

You can also send a message at a scheduled time in the future. See [Webex App | Send a scheduled message](https://help.webex.com/zphv2eb/) for more information.

When multiple images are shared in a message, you'll see them in a unified gallery view within the message on desktop devices.
