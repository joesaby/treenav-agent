# View Remote Access Reports for Your Webex Site

| 1 | Sign in to Webex Site Administration and go to Reports  \> Remote Access , and then select one of the following reports: Usage Report—Computer Usage Report—CSRs Tracking Report—Computer Computer Usage CSR Report Computer Tracking |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Specify the search criteria, and then click Display Report .                                                                                                                                                                          |
