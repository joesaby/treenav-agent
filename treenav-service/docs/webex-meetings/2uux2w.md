# Disable User Profile Pictures in the Meeting Application

The meeting application pulls profile pictures from Webex user profiles. If the user doesn't have an account or hasn't uploaded a profile picture, the meeting application uses a generic profile picture. You can disable unique profile pictures for the meeting application. 

| 1 | Sign in to Webex Site Administration and go to Configuration  \> Common Site Settings  \> Options .                                                                                                                                                    |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Scroll down to the Site Options  section.                                                                                                                                                                                                              |
| 3 | Uncheck Display user avatars in the meeting client (Meetings) . Generic profile pictures comprised of the first letters of the participant's first and last names appear for all participants. The color selection for each profile picture is random. |
