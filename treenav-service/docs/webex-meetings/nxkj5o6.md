# Disable Meeting Email Reminders in Cisco Webex Site Administration

This setting doesn't apply to meetings scheduled in Microsoft Outlook.

| 1 | Sign in to Webex Site Administration, and go to Configuration \> Common Site Settings \> Options. |
| - | ------------------------------------------------------------------------------------------------- |
| 2 | Scroll to Schedule Meeting Options, and under Meeting email reminders, select Off.                |
| 3 | Scroll to the bottom of the page and click Update.                                                |
