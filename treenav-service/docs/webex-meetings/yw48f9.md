# Delete a User's Recordings from Your Webex Site

| 1 | Sign in to Webex Site Administration and go to User Management  \> Edit User . |
| - | ------------------------------------------------------------------------------ |
| 2 | Locate the user account and click the link next to their user ID.              |
| 3 | Scroll down to the Recordings and Content Management  section.                 |
| 4 | Check the Recordings  check box, and then click Delete .                       |
| 5 | Confirm that you intend to delete all recordings associated with this account. |
