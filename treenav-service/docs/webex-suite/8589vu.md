# Webex App migration from teams.webex.com to web.webex.com

Webex App for web is migrating from teams.webex.com to [web.webex.com](https://web.webex.com). Web.webex.com is the product we're investing in as we go forward, adding features and providing fixes.

The new Webex App for web at [web.webex.com](https://web.webex.com) has been available for some time, and has been running without issue. 

The old Webex App for web at teams.webex.com has been running in parallel with the new one, but will be turned off in **late February 2022**

[Will I need a new Webex account?](#Cisco%5FReference.dita%5F5e605dc0-11bd-4013-91bb-6207f765eba0)

No. If you were using teams.webex.com, then your existing account will allow you to use [web.webex.com](https://web.webex.com)

[I have been emailed meeting join links. Will they still work?](#Cisco%5FReference.dita%5F68251674-e8ca-48bb-b3b2-01bef956e910)

Yes. Any meeting join links will bring you to the correct meeting.

[What happens if I continue to go to the old client?](#Cisco%5FReference.dita%5F1efc79c0-c98c-4626-ae43-cd48b57de840)

All traffic will be directed to [web.webex.com](https://web.webex.com).

[I have bookmarked the URL of my favorite space in my browser. Can I still use this URL?](#Cisco%5FReference.dita%5F68341f25-8fee-4907-92b1-0d44e2ba79af)

The URL can still be used, but it will take you to the main page of [web.webex.com](https://web.webex.com), instead of taking you to the specific bookmarked space.

[Does it work with Microsoft Internet Explorer?](#Cisco%5FReference.dita%5F9cb30239-17b7-4ed3-8c02-9da4946e689c)

No. While the old Webex App for web client has historically worked with Microsoft Internet Explorer, support was officially dropped on October 5th. See [Webex App | What's New](https://help.webex.com/en-US/article/8dmbcr) — Microsoft browser support is limited to the last two major releases of Microsoft Edge.

The new Webex App for web client does not work with Microsoft Internet Explorer.
