# Overview of One Button to Push (OBTP)

## Join button on Board, Desk, and Room series device

Any of these methods can be used by users to schedule meetings and incorporate Cisco devices featuring the Join button:

**Webex meetings**

- From a [Webex space, using the Webex app](https://help.webex.com/article/nu5w9x3).
- From the calendar, using [the Webex Scheduler for Microsoft Outlook](https://help.webex.com/en-us/article/pzti9s/Deploy-and-configure-the-Webex-Scheduler-for-Microsoft-Outlook).
- From the calendar, by entering [a scheduling keyword in the meeting location field](https://help.webex.com/article/nwtulzs).
- From the calendar, by entering [a supported video address format in the meeting location or body](https://help.webex.com/article/wt50w0/Meetings-with-Video-Addresses-in-Your-Cisco-Webex-Teams-Meetings-List).
- Using the [Webex Meetings Scheduler for Google Workspace](https://help.webex.com/article/n5mcx64).

To provide One Button to Push on Cisco devices, you need to [deploy the Hybrid Calendar](https://help.webex.com/en-us/article/niqovwv/Hybrid-Calendar-Service-with-Office-365-integration-reference). The details of the deployment depend on the type of calendar environment that you have and on the type of device.

## One button to push availability for different types of meetings

The following tables show the situations in which the Hybrid Calendar Service can provide OBTP to either premises or cloud-registered endpoints.

Hybrid Calendar runs on devices that can register to the Webex cloud by using RoomOS.

__Table 1\. OBTP for Meeting Services__
| Meeting type                                                                                                                 | TMS-managed with XE | Hybrid Calendar Service |
| ---------------------------------------------------------------------------------------------------------------------------- | ------------------- | ----------------------- |
| Webex                                                                                                                        | PT-scheduled only   | Yes                     |
| Personal Room meeting                                                                                                        | PT-scheduled only   | Yes                     |
| Cisco Meeting Server                                                                                                         | Yes                 | Yes                     |
| Microsoft Teams                                                                                                              | No                  | Yes                     |
| Google Meet                                                                                                                  | No                  | Yes                     |
| [Third-party](https://help.webex.com/en-US/article/wt50w0/Webex-App-%7C-Meetings-with-video-addresses-in-your-meetings-list) | No                  | Yes                     |

__Table 2\. Calendar platform by service__
| Calendar platform  | TMS-managed | Hybrid Calendar Service |
| ------------------ | ----------- | ----------------------- |
| Microsoft Exchange | Yes         | Yes                     |
| O365               | Yes         | Yes                     |
| Google Calendar    | No          | Yes                     |

__Table 3\. Device support__
| Device     | TMS-managed | Hybrid Calendar Service |
| ---------- | ----------- | ----------------------- |
| Room OS/CE | Yes         | Yes                     |
| CTS        | Yes         | No                      |
| MXP        | Yes         | No                      |
| C Series   | Yes         | No                      |
