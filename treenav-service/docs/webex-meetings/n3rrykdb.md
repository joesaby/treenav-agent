# Check for Webex Productivity Tools Updates for Mac

If you are able to receive Webex Productivity Tools updates, you can check to see if an update is available. From the Cisco Webex  menu on the Mac menu bar, select More  \> Check for Updates . 

The system checks to see if an new update is available. If no update is available, a message appears indicating that the latest version is installed. If a new update is available, you can choose to install it immediately.

**End of Support for Mac Productivity Tools** 

The final release for Productivity Tools for Mac will be 41.5 as [ Microsoft plans to block injection-based plugins on Mac ](https://support.microsoft.com/en-us/office/outlook-for-mac-%E2%80%93-planned-update-for-injection-based-plugins-dc6f9490-da7d-4d35-8793-835636b954d5) Outlook beginning in the Fall. Mac based Outlook customers who wish to use Webex should move to the [ Cisco Webex Scheduler ](https://help.webex.com/n2a359db/).

Webex Productivity Tools is a separate package from the Webex Meetings desktop app from 39.10\. Productivity Tools no longer supports integrations with IBM Lotus Notes, IBM Sametime, Microsoft Lync, Internet Explorer, and Skype for Business from 41.2.
