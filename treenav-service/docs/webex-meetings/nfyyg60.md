## Set User Account Privileges

Site administrators can set the privilege assignments for a user account at any time by specifying the following: 

- The types of meetings a user can host on a Webex service site. Meeting types determine which features a user can use on your site.
- Whether a user can use My Webex Standard or My Webex Pro features.
- The teleconferencing services a user can use during a meeting or training session.

Site administrators can change the privileges for either all user accounts simultaneously, or only specific user accounts. 

## Set Privileges for Individual Accounts

| 1 | On the navigation bar, select Users \> Add User or Edit User.                                                        |
| - | -------------------------------------------------------------------------------------------------------------------- |
| 2 | Locate the user account and select the corresponding link.                                                           |
| 3 | On the Add User or Edit User page, in the Privileges section, select the privileges you want to enable for the user. |
| 4 | Select Add or Update.                                                                                                |
