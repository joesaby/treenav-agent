# Webex App | Share files from Google Drive

- [Desktop](#Cisco%5FTask.dita%5F70f0480f-3482-44bc-a547-7cbdf38b51c8)
- [Mobile](#Cisco%5FTask.dita%5Fc6e00e68-891f-43eb-9aea-f5546c65afe2)

You can select a file directly from Google Drive and then choose who can view the file, after you share it in the space. 

There are two options for sharing files:

- Direct share using a file URL.
- Sharing using the Webex Attachment button, .  
When a file is shared across multiple messages with different permissions, the most recent permission settings apply to all instances of the shared file.

## Direct share using a file URL

You can now share links to files in a supported file sharing service (OneDrive/SharePoint, Box, Google Drive) simply by pasting a link into a Webex Space or direct message! 

Once you've shared the link, you'll have a file sharing experience similar to files shared using the paperclip icon:

- The message shows a thumbnail for the file with basic details as well as **Refresh** and **Copy share link** options.
- The content also appears in the **Files** tab of the space's **Content**tab.
- You can open the file directly within Webex rather than within an external browser.

To share a link as a file using ECM (Microsoft SharePoint/OneDrive, Box, or Google Drive), make sure that you meet the following prerequisites:

1. You must have ECM file sharing enabled.
2. The specific ECM service (Microsoft OneDrive/SharePoint, Box, or Google Drive) must be enabled for your organization by an administrator.
3. You must be logged in to the ECM service.  
If you're not logged in, you'll be prompted to do so. If you're not prompted to login because the service isn't enabled by your administrator, the file reference appears as a regular link.

## Share using the Webex Attachment button

1. In the message area, click Attachment , and then choose Share from Google Drive.
2. Select a file on Google Drive, then decide who can view the file in the space:  
  - Anyone—Anyone can view the file, including people from outside your organization.
  - People in your organization—Only people in your organization can view the file.
  - People with existing access—Webex App uses the permissions already set up for viewing the file in Box.  
When people don't have access to the file, they see the file name but they can't view the file or see a thumbnail preview.
3. Make your selection and click Apply.
4. Click  to send the link.
