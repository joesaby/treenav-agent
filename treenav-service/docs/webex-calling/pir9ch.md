# Manage application services for a user

| 1 | Sign in to [Control Hub](https://admin.webex.com).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Go to Management \> Users.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 3 | Select a user and click Calling.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 4 | Go to User call experience and click Application line Assignments.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 5 | Under Ring settings, choose the ringing behavior for the user's devices in these scenarios: **Click to Dial calls are made**—Notifies the user of incoming click-to-dial calls on the Webex Calling application. **Group Pages are received**—Notifies the user when group paging calls are made on the Webex Calling application. **A call is parked on this user's line**—Notifies the user when a call is parked on their line. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/448001-449000/448759.jpg)                                                                                                                                                     |
| 6 | Under Webex application assignments, choose the available devices that are licensed with the Webex Calling application for the user. Web Desktop Mobile Tablet If you’ve already added the maximum number of lines (35) to this user, you can’t select the options. All the Webex applications (Web, Desktop, Mobile, and Tablet) added to the user are counted as a single device. Users can't use the Webex Calling application on these devices unless you assign them a calling license. ![Webex application assignments section with the desktop, mobile, and tablet options checked](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/448001-449000/448760.jpg) |
| 7 | Click Save to confirm your changes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
