# Uninstall the Webex Meetings Desktop App

- [Windows](#task%5FBA7BE54214E2CA14984F10C1171FF47D)
- [Mac](#task%5FA250C166AD4B5967AC91192BF10EE2B5)

| 1 | On your computer, go to Start \> Settings \> Apps. |
| - | -------------------------------------------------- |
| 2 | Select Webex Meetings Desktop App.                 |
| 3 | Select Uninstall.                                  |
| 4 | Follow the prompts.                                |

| 1 | On your dock, select the Finder.          |
| - | ----------------------------------------- |
| 2 | Go to Applications \> Webex \> Uninstall. |
| 3 | Follow the prompts.                       |
