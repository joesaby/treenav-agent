| Go to Messaging ![Messaging button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/438001-439000/438646.jpg), select the space, and click People. Then click ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/429001-430000/429525.jpg) Add people, right-click the person's name, and select Assign as Moderator. If you want to remove someone as a moderator, right-click on the person's name and select Remove as Moderator. Non team members/guests can't moderate a space. |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| Select the space, then go to the activity menu ![More options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/461001-462000/461853.jpg) and choose People. Then tap the person's name and choose Assign as Moderator. If you want to remove someone as a moderator, tap on the person's name and select Unassign as moderator. Non team members/guests can't moderate a space. |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| Select the space, then go to the activity menu ![More Options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/453001-454000/453339.jpg) and choose People. Then tap the person's name and choose Assign as Moderator. If you want to remove someone as a moderator, tap on the person's name and select Unassign as moderator. Non team members/guests can't moderate a space. |

If you want to remove someone as a moderator, select More ![More Options button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/410001-420000/417001-418000/417371.jpg) beside the person's name and choose Remove as Moderator.
