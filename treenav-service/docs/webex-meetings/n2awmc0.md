# Enable Local Recordings in Control Hub

Hosts must join with VoIP for the audio to be recorded. Local MP4 recordings don't capture transcriptions, chat records, participant lists, or files shared during the meeting.

| 1 | Sign in to [Control Hub](https://admin.webex.com/), then go to Services \> Meeting \> Sites.                                                                                                                                                                                                                                                                                                                                                  |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the Webex site for which you want to enable local recordings, and click Configure Site.                                                                                                                                                                                                                                                                                                                                                |
| 3 | Under Common Settings, select Site Options.                                                                                                                                                                                                                                                                                                                                                                                                   |
| 4 | In the Site Options section, check Enable local recording for Webex meetings, events, training sessions, and Webex Teams meetings to allow users to save recordings directly to their computers. You can have a disclaimer appear before locally recording an MP4 meeting by checking the  Show disclaimer for local recording (MP4 only) check box and then entering a URL link to your organization's privacy guidelines in the text field. |
| 5 | Click Update.                                                                                                                                                                                                                                                                                                                                                                                                                                 |
