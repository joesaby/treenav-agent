# Start a Webex Instant or Personal Room Meeting by phone

Starting and managing your meetings from only your phone is possible when necessary, but it will provide limited meeting controls. Therefore, we recommend that you always start and manage your meeting from the Meetings desktop, web, or mobile app.

Your administrator must enable your Webex site to allow hosts to start Personal Room meetings from their phones.

| 1 | On the home page of your Webex site, click More ways to join. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/427001-428000/427767.jpg) The meeting information appears.![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/427001-428000/427768.jpg) |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Dial one of the phone numbers.                                                                                                                                                                                                                                                                                                                  |
| 3 | Enter the access code of the meeting and press #.                                                                                                                                                                                                                                                                                               |
| 4 | When prompted for an attendee code, press #.                                                                                                                                                                                                                                                                                                    |
| 5 | When prompted for the host PIN, enter the host PIN and press #. Attendees only press # to join. If the host has already started the Personal Room meeting, attendees automatically join; if it hasn't started yet, they wait in the lobby.                                                                                                      |
