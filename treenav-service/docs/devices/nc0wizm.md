# Hide meetings details on Board, Desk, and Room series devices

You can set your meeting details to private from the local web interface of the device. Read the [Device Configurations ](https://help.webex.com/en-us/n5pqqcm/Advanced-Configurations-for-Room-and-Desk-Devices-and-Webex-Boards) article for more information on how to access it.

When the web interface of the device is opened, navigate to the Setup  tab, select Settings , and choose the following configuration:

- Configuration  \> UserInterface  \> Bookings Visibility Title : Choose hidden  to hide the meetings titles. "Scheduled meeting" will be displayed.
