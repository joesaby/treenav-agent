# Set Webex Meetings as the Default for Video Conferencing in Google Calendar

## Turn Off Google Meet as the Default for Video Conferencing

If most people in your organization use Webex Meetings, but some need to use Google Meet sometimes, disable Meet as the default. Then, your users can choose their video conferencing provider each time.

| 1 | [Sign in](https://admin.google.com/) to your Google Admin console.          |
| - | --------------------------------------------------------------------------- |
| 2 | From the Admin console Home page, go to Apps \> GSuite \> Calender.         |
| 3 | Click Sharing settings \> Video conferencing.                               |
| 4 | Uncheck Make Google the default video conferencing provider when available. |
| 5 | Click Save.                                                                 |

## Set Webex Meetings as the Default Video Conferencing Provider

If everyone in your organization uses Webex Meetings, set it as the default video conferencing provider.

| 1 | [Sign in](https://admin.google.com/) to your Google Admin console.                                                                                                                                     |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | From the Admin console Home page, go to Apps \> GSuite.                                                                                                                                                |
| 3 | Next to Google Meet, click More options ![More options](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/443001-444000/443061.jpg) \> Turn OFF for everyone. |
| 4 | [Install the Cisco Webex app from the Google Workspace Marketplace](https://help.webex.com/4tzekz/#id%5F83219) for all users.                                                                          |
| 5 | From the Admin console Home page, go to Apps \> GSuite \> Calender.                                                                                                                                    |
| 6 | Click Sharing settings \> Video conferencing.                                                                                                                                                          |
| 7 | Check Automatically add video calls to events users create.                                                                                                                                            |
| 8 | Click Save.                                                                                                                                                                                            |
