# View Storage Usage Reports for Your Webex Site

When you generate a storage usage report, it includes usage for the following:

- MP4 recordings
- ARF recordings
- Uploaded recordings
- Disabled recordings
- Recordings in trash
- Chat in Webex Events recordings
- Uploaded files

| 1 | Sign in to Webex Site Administration and go to Reports  \> Common  \> Storage Usage .               |
| - | --------------------------------------------------------------------------------------------------- |
| 2 | To open the report in .CSV format, select Open . To save the report to your computer, select Save . |
