# Receive a Warning When Webex Site Storage Capacity Is Exceeded from Control Hub

Your Webex site provides a limited amount of storage space for user recording files in the My Files  area of My Webex. When the capacity for your site is exceeded, users cannot store more files on the site. You can set this option so that you receive a warning message about the storage capacity, the next time that you sign in to Control Hub.

| 1 | From the customer view in [https://admin.webex.com, ](http://admin.webex.com) go to Services .                        |
| - | --------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Meeting , select Sites , choose your site, and then select Configure Site .                                     |
| 3 | Select Site Options , then locate Notify site administrator if storage usage exceeds \_\_% of total allocated space . |
| 4 | Specify the percentage of total allocated space at which you want to receive a warning message.                       |
| 5 | Select Update .                                                                                                       |
