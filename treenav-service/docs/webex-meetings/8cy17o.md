# Enable a Webex Session Type for All New Users

## Enable a Session Type for all new users

| 1 | Sign in to Webex Site Administration and go to Configuration  \> Common Site Settings  \> Session Types .                                             |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Default for New Users  column, locate and check the box that corresponds with the session type to which you want all new users to have access. |
| 3 | Click Update .                                                                                                                                        |
