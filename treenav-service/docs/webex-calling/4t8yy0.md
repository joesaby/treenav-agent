# Report all phone issues for your Cisco MPP device

You can use the Problem Reporting Tool (PRT) to collect and send phone logs, and to report problems to your administrator.

Before you begin

Your administrator enables web server for the phone. 

| 1 | Press Applications ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381337.jpg).                                                                                                                                                              |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Press Settings.                                                                                                                                                                                                                                                                                           |
| 3 | Do one of the following actions: Desk phones—Press Applications ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381337.jpg). Conference phones—Press Settings.                                                                               |
| 4 | Select Status \> Report problem.                                                                                                                                                                                                                                                                          |
| 5 | Enter the date that you experienced the problem in the Date of problem field. The current date appears in this field by default.                                                                                                                                                                          |
| 6 | Enter the time that you experienced the problem in the Time of problem field. The current time appears in this field by default.                                                                                                                                                                          |
| 7 | Select Problem description.                                                                                                                                                                                                                                                                               |
| 8 | Select a description from the displayed list.                                                                                                                                                                                                                                                             |
| 9 | Press Submit. If your administrator disables web server for the phone, the phone fails to upload the problem report to the server. In addition, the phone screen shows Error: 109 or Report Problem together with an unavailable download link. Contact your administrator to help you resolve the issue. |
