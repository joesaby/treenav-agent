# Reassign Webex Programs or Categories in Cisco Webex Control Hub

Site administrators can reassign Webex Events programs or Webex Training categories to a different owner if that person leaves an organization.

| 1 | From the customer view in [https://admin.webex.com ](https://admin.webex.com/login), go to Users , and select the user you want to reassign programs or categories for. |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Services,  select Meeting .                                                                                                                                       |
| 3 | Choose the Webex site containing the user's settings, and then select User Privileges .                                                                                 |
| 4 | Click the Advanced Settings  link, and scroll to the Recording and Content Management  section.                                                                         |
| 5 | Check the boxes for the content types to reassign: Webex Events programs Webex Training categories                                                                      |
| 6 | Select the Reassign  link, and use the Search  feature to locate the account to which to reassign the content to.                                                       |
| 7 | Click Update .                                                                                                                                                          |
