# Disable meeting email reminders in Control Hub

This setting doesn't apply to meetings scheduled in Microsoft Outlook.

| 1 | Sign in to [Control Hub](https://admin.webex.com/), then go to Services.                                              |
| - | --------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Meeting, select Sites.                                                                                          |
| 3 | Select the Webex site that you want to change the settings for, and then click Settings.                              |
| 4 | Under Common Settings, select Schedulerand uncheck Allow email reminders for meetings, training sessions, and events. |
