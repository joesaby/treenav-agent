# Add Licenses for speech commands for Webex Meetings

Before you begin

Speech commands are available in Webex Meetings. Beginning with the 41.5 update, it's also available in Webex Webinars for plans that support up to 3000 users and as an add-on for plans that support up to 1000 users. 

| You can assign speech commands for Meetings licenses in the following ways: OptionDescription Add a license for an existing user Follow the instructions in [Edit service licenses in Control Hub for individual users](https://help.webex.com/9dmqgv/). Add licenses for multiple users Follow the instructions in [Add multiple users in Control Hub with the CSV template](https://help.webex.com/e2okky/). Add licenses for new users, by default Follow the instructions in [Set up automatic license assignments in Control Hub](https://help.webex.com/n3ijtao/). |
