# Receive a Warning When Webex Site Storage Capacity Is Exceeded

Your Webex site provides a limited amount of storage space for user recording files in the My Files  area of My Webex. When the capacity for your site is exceeded, users cannot store more files on the site. You can set this option so that you receive a warning message about the storage capacity, the next time that you sign in to Site Administration. 

| 1 | Sign in to Webex Site Administration and go to Configuration  \> Common Site Settings  \> Options .                      |
| - | ------------------------------------------------------------------------------------------------------------------------ |
| 2 | In the Site Options  section, locate Notify site administrator if storage usage exceeds \_\_% of total allocated space . |
| 3 | Specify the percentage of total allocated space at which you want to receive a warning message.                          |
| 4 | Select Update .                                                                                                          |
