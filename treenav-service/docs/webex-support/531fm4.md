# Manage Webex Support Session Forms in Cisco Webex Site Administration

##  Webex Support Session Forms

Before and after sessions, site administrators can have Webex Support ask users to provide information: 

- Pre-Session forms that customers can use before joining sessions to provide contact information or to ask questions.
- Post-Session forms that customers can use when leaving sessions to provide comments about the service.

## Customize Items on the Pre-Session and Post-Session Forms

| 1  | Sign in to Webex Site Administration and go to Configuration \> Webex Support \> Forms.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| -- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2  | Choose whether you want to customize a Pre-Session or Post-Session form.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 3  | Check the box in the first column for each item you want to appear on the form, and then check the box in the second column for each item you want to require that participants complete.                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 4  | To add a new item to the form, select Add New.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| 5  | To add a text box where users can enter information: Select Text box. To create a text box that is more than one line, select Multi-line. At Text box label, enter the name that you want to appear next to the box. At Width, enter the width of each line in the box. Select Save.                                                                                                                                                                                                                                                                                                                                                                                   |
| 6  | To add an option box that users can use to select a single option: Select Check boxes. At Group label, enter any text that you want to appear before the options. At Check box, enter the name that you want to appear for each option that users can select. If you want an option to appear selected, choose Selected from the list at the left. Repeat steps c and d for each additional option you want to provide on the form. To add more options, select the Add drop-down list and select the number of options you want to add. Select Save.                                                                                                                  |
| 7  | To add a check box that users can use to select one or more options: Select Options buttons. At Group label, enter any text that you want to appear before the options. At Choice, enter the name that you want to appear for each option that users can select. If you want an option to appear selected, choose Selected from the list at the left. To make one of the choices the default option, select which option from the Default choicedrop-down list. Repeat steps c and d for each additional option you want to provide on the form. To add more options, select the Add drop-down list and select the number of options you want to add. Select Save.     |
| 8  | To add a drop-down list that hat users can use to select a single option: Select Drop-down list. At Group label, enter any text that you want to appear before the options. At Choice, enter the name that you want to appear for each option that users can select. If you want an option to appear selected, choose Selected from the list at the left. To make one of the choices the default option, select which option from the Default choicedrop-down list. Repeat steps c and d for each additional option you want to provide on the form. To add more options, select the Add drop-down list and select the number of options you want to add. Select Save. |
| 9  | To change the order of the items on the list, from the form page: Select Change Order. Select the item you want to move and then use the up and down arrows to move it. Select Save.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 10 | Select Save.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

## Embed the Pre-Session Form

Use this procedure to generate HTML code that you can use to embed the Pre-Session form to your web page or a button that links to it. 

 You can choose from several types of buttons that users select to display the Pre-Session form. 

| 1 | Sign in to Webex Site Administration and go to Configuration \> Webex Support \> Forms.                                                                                                                                                                                                                                                                                                                                                                               |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Pre-Session.                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 3 | Select the Embed Form tab.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 4 | Select one of the following: Embed the form within any of your own web pages: to generate the HTML code for the Pre-Session form. Embed a button linking to form on any of your own web pages: to generate the HTML code for a link to the Pre-Session form.                                                                                                                                                                                                          |
| 5 | If you are creating a button, select the type of button that you want to create in the Default Buttons section.                                                                                                                                                                                                                                                                                                                                                       |
| 6 | Select Generate HTML to display the HTML code in the text box.                                                                                                                                                                                                                                                                                                                                                                                                        |
| 7 | Select Preview to show the Pre-Session form or your selected button.                                                                                                                                                                                                                                                                                                                                                                                                  |
| 8 | Select either Go Back or the Forms tab and then select Save. The Pre-Session form always includes a support session number that Webex Support automatically creates for each support session. You cannot remove this number from the form. The information that you obtain in a Pre-Session or Post-Session form is stored in the support session’s log. You can view the log and export the data into a file that you can open in a spreadsheet or database program. |
