# Webex App | Send a Space Link

- [Desktop](#Cisco%5FTask.dita%5F28cd0186-dddd-4f78-89ed-77b832024105)
- [Mobile](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5F096a73fe-b923-42b3-bf55-e8a3a801989b)

| In a space, type @ and then select the space name from the list. Then click Send![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/437001-438000/437489.jpg). You can also send a space link. In the space that you want to share, click Space Information ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/427001-428000/427579.jpg) and then select Copy Space Link. Space links can only be shared in other Webex App spaces. |
