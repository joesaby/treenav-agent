# Assign WebACD Queues to Agents on Your Webex Site

You can also select queues for WebACD on the Add User page, under WebACD Preferences.

| 1 | Sign in to Webex Site Administration and go to Users  \> Edit User .                                                                                                                                                                                                                             |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Scroll down to WebACD Preferences  and select the check boxes to indicate whether the user is an agent or a manager, or both.                                                                                                                                                                    |
| 3 | Select Select Queues  and then check the check box for each queue that you want to assign. Alternately, to assign all queues, check the Automatically assign all Webex Meeting queues  check box.                                                                                                |
| 4 | To assign Webex Support queues, select Select Queues  and then check the check box for each queue that you want to assign. Alternately, to assign all queues, check the Automatically assign all Webex Support queues  check box.                                                                |
| 5 | To assign agents to managers, select Select Agents , and then check the check box for each agent that you want to assign. Alternately, you can check either or both of the following check boxes: Automatically assign all Webex Meetings agents. Automatically assign all Webex Support agents. |
| 6 | Select Update .                                                                                                                                                                                                                                                                                  |
