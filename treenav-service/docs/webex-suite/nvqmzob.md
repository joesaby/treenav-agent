# Cisco Collaboration Flex Plan Shared Meetings

When you purchase the Cisco Collaboration Flex Plan Shared Meetings subscription, you can do the following: 

- When you assign the shared meetings license to people in your organization either individually or using the CSV file. Each subscription allows one concurrent meeting to be hosted by anyone with the shared meetings license.
- To review your license usage for the subscription, see [View License Usage in Cisco Webex Control Hub](https://help.webex.com/article/nkwcu4bb).
- To view the shared meetings reports, see [Review the Cisco Collaboration Flex Plan Shared Meetings Report](https://help.webex.com/article/78s491).

For more information on Cisco Collaboration Flex Plan, see <http://www.cisco.com/c/en/us/products/unified-communications/spark-flex-plan/index.html>.
