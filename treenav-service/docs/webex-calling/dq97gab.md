# Use Push to Talk on Your DECT Handset

You can use the handset as a two-way radio with the Push to Talk feature. The message is played over the handset speaker automatically. Only 1 person at a time can use Push to Talk. If you press Mute while you are receiving a Push to Talk message, the handset speaker mutes.

Before you begin

- You need Push to Talk enabled on your handset.
- At least one more handset in the system needs Push to Talk enabled.
