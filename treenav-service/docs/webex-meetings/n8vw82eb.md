# Webex App | Capacities

## Teams

| Features               | Maximum Limit |
| ---------------------- | ------------- |
| Teams per organization | Unlimited     |
| Team members           | 5000          |

## Spaces

| Features                | Maximum Limit                                                   |
| ----------------------- | --------------------------------------------------------------- |
| Spaces per organization | Unlimited                                                       |
| Spaces per team         | Unlimited                                                       |
| Space image             | File size: 10 MB or less. File types: .jpg, .png, .gif, or .bmp |
| Space participants      | 5000                                                            |
| Messages per space      | Unlimited                                                       |

## Files and Whiteboards

| Features                 | Maximum Limit                                                                                                                       |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| Whiteboards per space    | Unlimited                                                                                                                           |
| Files per space          | Unlimited                                                                                                                           |
| Preview files in a space | File size: 100 MB or less File has less than 50 pages File type: .doc, .docx, .gif, .jpeg, .pdf, .png, .ppt, .pptx, .xls, and .xlsx |

## Filters

| Feature          | Maximum Limit |
| ---------------- | ------------- |
| Favorite spaces  | Unlimited     |
| Mentions listed  | 50            |
| Flagged messages | 200           |
| Draft messages   | Unlimited     |

## Message

| Feature                                          | Maximum Limit                                               |
| ------------------------------------------------ | ----------------------------------------------------------- |
| Message size per post                            | 5400 characters                                             |
| Number of files you can add to a single message. | 10 files on Windows, Mac, iOS, and Web. 5 files on Android. |
| File size                                        | 2GB                                                         |
| Auto play GIF                                    | 2MB                                                         |

## Profile Picture

| Feature   | Maximum Limit                                                                                                                                                                                  |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| File size | 10 MB or less Webex App resizes your profile picture to use in various areas in Webex App. Ideally, your profile picture should be a square image between 200x200 pixels and 1000x1000 pixels. |
| File type | .jpg, .png, .gif, or .bmp                                                                                                                                                                      |

## Meetings

| Feature                                                                           | Maximum Limit |
| --------------------------------------------------------------------------------- | ------------- |
| Meetings - Free or messaging-only Webex App accounts                              | 100           |
| Meetings - Paid Webex Calling or Webex App account                                | 150           |
| Meetings - Webex Meetings host account or a basic meetings (M2) Webex App license | 200           |

## Data Storage

Data storage is based on the Cisco Webex plan and retention policy for your organization. For information, see [Cisco Webex Plans and Pricing ](https://www.webex.com/pricing/index.html) or view your organization's policy for spaces [Space Policy ](https://help.webex.com/article/baztm6).
