# Check for Webex Productivity Tools Updates for Windows

If your site administrator allows you to receive automatic updates of Webex Productivity Tools, you can check to see if an update is available. 

| In Microsoft Outlook, go to Schedule Meeting \> More \> Check for Updates.  Productivity Tools checks to see if a new update is available. If no update is available, a message appears indicating that the latest version is installed. If a new update is available, you can choose to install it immediately. When Microsoft Outlook is open, Webex Productivity Tools checks for updates automatically every two hours. |
