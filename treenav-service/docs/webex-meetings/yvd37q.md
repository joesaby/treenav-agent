# Update Webex Productivity Tools

If your site administrator leaves the Automatically upgrade Webex Productivity Tools when new versions are available unchecked in Webex Site Administration or Webex Control Hub, then Productivity Tools will stay on version 39.10\. IT administrators will then have to deploy the latest version of Productivity Tools on a regular basis.

- [Windows](#id%5F128009)

| 1 | If your site is set to automatically update to the latest version, then users will see a red dot visible on Schedule Meeting. Restarting Microsoft Outlook will update Productivity Tools automatically. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/441001-442000/441416.jpg)                                 |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | To manually check for updates, from the Home ribbon in Microsoft Outlook, click Schedule Meeting  \> More \> Check for Updates in the Webex group. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/441001-442000/441683.jpg) If there is an update available, there will be a red dot visible on Schedule Meeting. |
| 3 | Click Check for Updates. You can click Update or wait for it to automatically update to the latest version. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/441001-442000/441687.jpg)                                                                                                                              |
