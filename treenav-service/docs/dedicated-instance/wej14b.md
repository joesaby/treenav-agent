# Find the Data Residency Region That Maps to a Country

Use these steps to check the region (Europe or North America) to which a particular country maps.

| 1 | Download [CountryCodeGEOMapping.xlsx](https://www.cisco.com/c/dam/en/us/td/docs/voice%5Fip%5Fcomm/cloudCollaboration/wbxt/datalocality/CountryCode/CountryCodeGEOMapping.xlsx).                                                      |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Open the file in Microsoft Excel.                                                                                                                                                                                                    |
| 3 | Choose from one of the following actions: On the User page tab, select a country from the drop-down list in cell B3\. The associated region is displayed in cell B4.Use the Country Code GEO mapping tab to browse the country list. |
