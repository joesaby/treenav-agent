# Start a Webex Meeting from your Webex site

## Start a Personal Room meeting

| 1 | From your Personal Room in your Webex site, click the drop-down arrow and select Use desktop app or Use web app, then select Start a Meeting. |
| - | --------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | After the meeting opens, select your [audio and video options](https://help.webex.com/ela6i8/).                                               |

## Start a scheduled meeting

| 1 | Select Meetings from the left navigation panel in your Webex site, and click Start ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/439001-440000/439463.jpg) next to the meeting. Your meeting opens in either the desktop app or web app, depending on which option you last chose. |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | To choose which app to use, click the name of the meeting from the meetings list, then click the drop-down arrow and select Use desktop app or Use web app. Both have a similar meeting experience, but the web app only requires your browser. After that, select Start Meeting.                                                   |
