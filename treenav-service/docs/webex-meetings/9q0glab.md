# Manually Install Webex Meetings for Mozilla Firefox

This document contains instructions on how to manually install the Webex Meetings extension in Mozilla Firefox.

**To manually install the Webex Meetings extension in Firefox:** 

1. Download the archive: **`Cisco_Webex_Extension.xpi.zip `** (see link below).
2. Locate the downloaded zip file and open it. Save the extension file **`Cisco_Webex_Extension.xpi `** to your desktop or other convenient location.
3. Open a new tab in Firefox and drag **`cisco_webex_extension.xpi `** into the browser.
4. A dialog box will open and ask for your permission. Select Add .
5. Click Install  and, if required, restart Firefox.

You can now join your meeting.

Your computer may still have to automatically download and install a Webex Meetings add-on file when you first join a meeting.

Download [The Cisco Webex for Firefox extension ](http://www.cisco.com/c/dam/en/us/td/docs/collaboration/webex%5Fcenters/esp/cisco%5Fwebex%5Fextension%5Fxpi.zip).
