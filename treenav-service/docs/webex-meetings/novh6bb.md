# Manage Content Sharing Security on Your Webex Site

## 

- [Site Administration](#id%5F137241)
- [Control Hub](#id%5F137242)

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Scroll down to Site Options \> Webex Meetings Content Sharing Security and configure the following options: Check or uncheck Allow screen capture (Android devices only). This feature applies only to Webex; users can still take screen captures while multitasking outside of the Webex app. Check or uncheck  Allow users to automatically accept all requests for keyboard and mouse control.  When the option is unchecked, the presenter has to manually pass control to another participant. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/451001-452000/451349.jpg) |

| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |

| 3 | Under Common Settings, select Site Options.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 4 | Scroll down to Webex Meetings Content Sharing Security and configure the following options: Check or uncheck Allow screen capture (Android devices only). This feature applies only to Webex; users can still take screen captures while multitasking outside of the Webex app. Check or uncheck Allow users to automatically accept all requests for keyboard and mouse control.  When the option is unchecked, the presenter has to manually pass control to another participant. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/451001-452000/451349.jpg) |
