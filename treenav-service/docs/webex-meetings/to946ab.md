# Show Custom Toll-Free Dialing Restrictions in Cisco Webex Control Hub

| 1 | From the customer view in [https://admin.webex.com ](https://admin.webex.com/login), go to Services , and under Meeting , select Sites . |
| - | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Choose the Webex site to change the settings for, and click Configure Site .                                                             |
| 3 | Under Common Settings , select Site Options .                                                                                            |
| 4 | From the Toll-free dialing restrictions  drop-down list, select Customer toll-free dialing restrictions .                                |
| 5 | To show customer toll-free dialing restrictions, select Yes .                                                                            |
| 6 | To provide the document on a web page, select URL , and then enter the URL in the corresponding field.                                   |
| 7 | To provide a PDF version of the document, select PDF file , and then click Upload File .                                                 |
| 8 | Browse to the file location, select the file, and then click Open .                                                                      |
| 9 | Select Update .                                                                                                                          |
