# Advanced customization of audio on Room devices

The Audio Console application lets you define how to connect audio inputs to outputs, using simple drag and drop techniques.

This utility is available for the following products:

- Codec Pro (Room Kit Pro)
- Room 70 G2, Room 70 Panorama, and Room Panorama
- Codec EQ (Room Kit EQ) and Room Kit EQX with AVIntegrator option

The main features of the Audio Console are:

- Configure the audio system of the codec.
- Change the default mixing, routing, and equalizers.
- Set various input and output connector properties.
- Configure advanced features such as local reinforcement.
