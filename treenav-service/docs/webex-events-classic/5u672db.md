# Manage Programs in Events (classic)

## Create Program

A program allows attendees to conveniently register for all its events at one time. All the events in a program share the same registration form. 

You can modify the programs you created on your Webex site at any time. 

| 1 | Sign in to your Webex site and select Webex Events (classic).                                                                                                                                                                                                                                                           |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Do any of the following: On the left navigation bar, select Create New Program. On the Schedule an Event page or Edit Event page, select Add a new program, which adds the event.                                                                                                                                       |
| 3 | Specify the options on the page, noting the following: If the e-commerce feature is turned on, you can require payment for an event by typing an amount in the Event fee box. If the promotion codes feature is turned on and you offer discounts for event fees, make sure that promotion codes are sent to attendees. |
| 4 | Select Create a Program.                                                                                                                                                                                                                                                                                                |
| 5 | If you created the new program from the Schedule an Event or  Edit Event page, select OK in the message box, informing you that adding this event changes its registration options to be the same as those of the program.                                                                                              |

## Add an Event to a Program

All events in a program share the same registration form. Attendees can choose to register for all the events in a program at one time. 

You can also add an event to a program while scheduling or editing an event. 

Before you begin

 After you add an event to a program, you can no longer modify registration approval rules.

| 1 | Sign in to your Webex site and select Webex Events (classic).                           |
| - | --------------------------------------------------------------------------------------- |
| 2 | On the left navigation bar, select List of Programs.                                    |
| 3 | Select the title of your program.                                                       |
| 4 | Select whether you want to add a new event or an event that has already been scheduled. |
| 5 | Follow the prompts to complete your action.                                             |

## Add a Recorded Event to a Program

If you have recorded events that you hosted, you can add them to a program. 

| 1 | Sign in to your Webex site and select Webex Events (classic).               |
| - | --------------------------------------------------------------------------- |
| 2 | On the left navigation bar, select List of Programs.                        |
| 3 | Select the title of your program.                                           |
| 4 | Select Add other recordings to this program.                                |
| 5 | Select the recordings that you want to add to the program, then select Add. |
| 6 | Select Continue.                                                            |

## Obtain URLs for Your Event, Program, or Recording

| After you sign in to your Webex site, do one of the following: To obtain an event URL, go to Meetings then select the name of the event to view the Event Information page. To associate source IDs, use the event address for attendees. To obtain a program URL, go to Webex Events (classic) and select List of Programs. Select the name of the program to see the Program URL. To obtain a recording URL, go to Webex Events (classic) and select My Event Recordings. Select the name of the recording to see the Playback/Download link. |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

## Edit a Program

| 1 | Sign in to your Webex site and select Webex Events (classic).                                                |
| - | ------------------------------------------------------------------------------------------------------------ |
| 2 | On the left navigation bar, select List of Programs. The programs you created have check boxes next to them. |
| 3 | Select the title of the program that you want to modify.                                                     |
| 4 | Select Edit program.                                                                                         |
| 5 | Make your changes, and then select Update program. You receive an automatic email message about this update. |

## Delete a Program

You cannot delete programs created by other hosts. 

| 1 | Sign in to your Webex site and select Webex Events (classic).                                                                                  |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | On the left navigation bar, select List of Programs. The Manage Programs page appears. The programs you created have check boxes next to them. |
| 3 | Check the check box for your program and select Delete.                                                                                        |
