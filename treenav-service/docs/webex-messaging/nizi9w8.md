# Webex App | Set notifications for specific spaces

Webex App allows you to set notifications in two ways: globally so that the settings apply to all of your spaces or individually so the settings apply just to certain spaces. These settings work together to make sure you get notified about the messages that matter most to you. See [Tips for Managing Notifications](https://help.webex.com/ng7w8cj/) for help choosing the best combinations.
