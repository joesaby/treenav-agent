# Display a banner ad for a Webex Meetings site in Control Hub

The banner ad displays at the top-right corner of a Webex Meetings site home page.

| 1 | Sign in to [Control Hub](https://admin.webex.com). |
| - | -------------------------------------------------- |
| 2 | Go to Services \> Meetings and select Sites.       |
| 3 | Select the Webex site and click Configure Site.    |
| 4 | Under Common Settings \> Site Options.             |
| 5 | Uncheck Display banner ad in My Webex.             |
| 6 | Click Update.                                      |
