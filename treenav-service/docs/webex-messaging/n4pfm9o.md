# Webex App | Screen reader support

| Platform        | Screen Reader                                                                                                                                                                                                                                           |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Windows         | You can use the two latest release of JAWS screen reader versions JAWS 2022 and JAWS 2023.                                                                                                                                                              |
| Mac             | You can use the built-in screen reader on your Mac. To enable, go to  System Settings \>  Accessibility  \>  VoiceOver .                                                                                                                                |
| iPhone and iPad | You can use the built-in iOS screen reader. To enable the screen reader, go to Settings  \> General  \> Accessibility  \> VoiceOver .                                                                                                                   |
| Android         | You can use the built-in screen reader on your Android device.                                                                                                                                                                                          |
| Web             | When you're using Webex App for web on Windows you can use the JAWS screen reader version JAWS 2022 or JAWS 2023 and when you're on Mac, you can use the built-in screen reader. To enable, go to  System Preferences  \>  Accessibility  \>  VoiceOver |
