# Webex App | Stop Text from Appearing in Message Notifications

| 1 | Tap your [profile picture ](https://collaborationhelp.cisco.com/article/m67xht), and then select Settings  \> Notifications  \> Messaging . |
| - | ------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Turn off the Show Message Preview  toggle.                                                                                                  |
