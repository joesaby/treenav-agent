# Transfer and Download Files During a Webex Training Session

## Publish Files During a Training Session

During a training session, you can publish files that reside on your computer, and training session attendees can download the files to their computers or local servers. 

| 1 | In the Session window, select File > Transfer. The File Transfer window appears.                                                                                                                                                                                                                                                                               |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Share File. The Open dialog box appears.                                                                                                                                                                                                                                                                                                                |
| 3 | Select the file that you want to publish then select Open. The file appears in the File Transfer window. The file is also now available in each attendee's File Transfer window.                                                                                                                                                                               |
| 4 | (Optional) Publish more files that you want attendees to download.                                                                                                                                                                                                                                                                                             |
| 5 | To stop publishing files during a training session, in the title bar of the File Transfer window, select the Close button. Training Manager closes the File Transfer window in each attendee's Session window.  The number of attendees that have the File Transfer window open, including you, appears in the lower-right corner of the File Transfer window. |

## Download Files During a Training Session

If a presenter publishes files during a training session, the File Transfer dialog box automatically appears in your Session window. You can then download the published files to your computer or a local server. 

| 1 | In the File Transfer window, select the file that you want to download.                                                                                                                                                                                          |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Download. The Save As dialog box appears.                                                                                                                                                                                                                 |
| 3 | Choose a location at which to save the file.                                                                                                                                                                                                                     |
| 4 | Select Save. The file downloads to your selected location.                                                                                                                                                                                                       |
| 5 | If applicable, download more files.                                                                                                                                                                                                                              |
| 6 | Once you finish downloading files, in the title bar of the File Transfer window, select the Close button.  To reopen the File Transfer window at any time, select File > Transfer. This option is available only if the presenter is currently publishing files. |
