# Set Board, Desk and Room Series devices to broadcast mode

Broadcast mode is aimed at broadcasting and recording services where you want to display only the video to your viewers. They won't see the icons, call controls, and notifications that are typically meant only for the persons in the room.

You can set a device to broadcast mode by accessing the local web interface. Read the [Advanced Settings ](https://help.webex.com/en-us/n5pqqcm/Advanced-Settings-for-Room-and-Desk-Devices) article for more information on how to access it. 

First, select Settings  from the menu on the left. Under Configurations , choose the following: 

| UserInterface  \> OSD  \> Mode . Select Unobstructed . Set to Auto  (default), the device will run in normal mode, and indicators, notifications, and controls will be included in the video stream. |
