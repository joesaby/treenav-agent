# Prevent Hosts from Uploading Recordings

- [Site Administration](#task%5F8A319B8345AE82160392DC5CD5FDFA91)
- [Control Hub](#task%5FCD501AAC731E9D4C7CF518C9EB8BF7F3)

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Options.                                                                                                                                                                                                                                                                                |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Scroll to Site Options, and uncheck Allow hosts to upload recordings. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/460001-461000/460469.jpg) The Add button no longer appears on the My Recorded Meetings page for hosts. To restore the button and the ability to upload recordings, check Allow hosts to upload recordings. |

| 1 | From the customer view in [ https://admin.webex.com, ](http://admin.webex.com) go to  Services .                                                                                                                                                                                                                                                                                                         |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Meeting, select Sites, choose your site, and then select Configure Site.                                                                                                                                                                                                                                                                                                                           |
| 3 | Under Common Settings, select Site Options, and then uncheck Allow hosts to upload recordings. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/460001-461000/460469.jpg) The Add button no longer appears on the My Recorded Meetings page for hosts. To restore the button and the ability to upload recordings, check Allow hosts to upload recordings. |
