# Reply to or vote for a question in a Q&A session in Slido

Slido is available in Webex Meetings and Webex Webinars. You can use Slido in Webex Meetings on [version 41.6 and later sites](https://help.webex.com/qgij81/), and in Webex Webinars on [version 41.9 and later sites](https://help.webex.com/qgij81/) and in Webex Suite meeting platform. Slido polls and Q&A in webinars are available for up to 10,000 attendees, while quizzes are available for up to 5,000 participants.

| 1 | Select Reply in the Q&A panel, type your reply, and then select Send. You'll only see Reply if the host has allowed meeting or participants to answer questions.                                                                                                                                                                                                                                                                                                                                       |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Select ![Thumbs up](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/459001-460000/459187.jpg) or ![Thumbs down](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/459001-460000/459188.jpg) to vote for the question. Voting ![Thumbs down](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/459001-460000/459188.jpg) is only available if enabled by the meeting host. |
