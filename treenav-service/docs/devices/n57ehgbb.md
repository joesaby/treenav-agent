# Group devices with tags in Control Hub

You can use tags to name a group of devices based on a business unit or a physical location. For example, if you tag a device with _third floor_, and then search for _third floor_, all devices with that tag show up.

Tags can also be used to amend the device name in the Webex App to help users to identify the correct device to connect to. For this purpose, use the format "name=<tag>". The first tag using this format gets added to the device name in parentheses in the Webex App connection menu.

| 1 | From the customer view in <https://admin.webex.com>, go to Devices, and select the device you want to tag. If you want to add or edit tags on multiple devices at once, select several devices and click on Edit in the toolbar.                         |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Tags in the panel that opens on the right, add a tag, and click outside the field to apply the tag. The tag doesn't appear in the device list, only on the device details page. However, you can use tags in a filter list to search for devices. |
