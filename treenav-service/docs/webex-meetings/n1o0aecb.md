# Configure Disclaimer Options for Your Webex Site in Cisco Webex Site Administration

## Set up Disclaimers

Set up your own customized disclaimers to appear each time that:

- A host starts a meeting.
- Participants join a meeting.
- Viewers start a recorded Webex session.

You can set up and control each type of disclaimer independently. If enabled, users must positively acknowledge these disclaimers to start or attend a meeting; or to join a session, or to watch a session recording.

| 1 | Use a text editor to create a disclaimer file and save the file. The following file types are supported: HTML, HML, and TXT. Create a disclaimer file for each type of disclaimer, and for each language that your site supports. |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Disclaimers.                                                                                                                              |
| 3 | On the Disclaimers page, select Upload Disclaimer.                                                                                                                                                                                |
| 4 | In the File Upload window, navigate to the file location on your computer, select the file, and then click Open.                                                                                                                  |
| 5 | In the Type column, from the list, select the corresponding type for the disclaimer.                                                                                                                                              |
| 6 | In the Language column, from the list, select the corresponding language for the disclaimer.                                                                                                                                      |
| 7 | In the Apply to Center(s) column, check the check boxes for the services, for which you want to display the disclaimer. MC—Webex MeetingsEC—Webex EventsTC—Webex TrainingSC—Webex Support                                         |
| 8 | To preview the disclaimer, select the filename. Alternately, in the Actions column, select the corresponding More button and then select Preview. The disclaimer opens in a new browser window.                                   |
| 9 | Select Save.                                                                                                                                                                                                                      |

## Delete a Disclaimer

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Disclaimers. |
| - | ---------------------------------------------------------------------------------------------------- |
| 2 | In the Actions column, select the corresponding More button, and then select Delete.                 |
| 3 | Click OK to delete the file.                                                                         |

## Disable a Disclaimer

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Disclaimers.                                                                                            |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Actions column, select the corresponding More button, and then select Disable. The disclaimer and corresponding configuration options are not available until you enable the disclaimer. |

## Enable a Disclaimer

| 1 | Sign in to Webex Site Administration and go to Configuration \> Common Site Settings \> Disclaimers. |
| - | ---------------------------------------------------------------------------------------------------- |
| 2 | In the Actions column, select the corresponding More button, and then select Enable.                 |
