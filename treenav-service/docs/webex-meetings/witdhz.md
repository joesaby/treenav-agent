# Changes to Toll-Free Call-In Numbers for Mexico

Due to local telecommunications regulatory changes, toll-free call-in numbers for Mexico for the Webex telephony bridge servicing your Webex site will be changed. This number change will be implemented during the following time window, based upon your site’s region:

| US EMEA APAC ANZ | August 16, 2019 between 7:00 PM and 11:59 PM PDT (GMT -7) August 17, 2019 between 7:00 PM and 11:59 PM GMT August 17, 2019 between 9:30 PM and \*2:30 AM (\*8/18/2019) Singaporte Time (GMT +8) August 17, 2019 between 9:00 PM and \*2:00 (\*08/18/2019) PM AEST (GMT +10) |

This update will only affect customers using Webex direct PSTN audio services, and only affects the Mexico toll-free call-in numbers. Customers using Cloud Connected Audio services are unaffected by this change.

All meetings will automatically display the new number for Mexico in the global call-in list. 

Any previously scheduled meetings that included the toll-free call-in number for Mexico in the meeting invite will need to be manually updated by the host to show the new number.
