# Change a Webex Custom Session Type

## Change a custom Session Type

| 1 | Sign in to Webex Site Administration and go to Configuration  \> Common Site Settings  \> Session Types .                                 |
| - | ----------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Session Code  column, locate the primary session type and then click the link for the custom session type that you want to edit.   |
| 3 | Locate the corresponding section for the primary session type, and then click the link for the custom session type that you want to edit. |
| 4 | Change the feature settings for the session type.                                                                                         |
| 5 | Select Update .                                                                                                                           |
