# Export a site configuration audit log

- [Control Hub](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5Fe74d40c1-52a1-4a8d-9721-31248cd7e23b)
- [Site Administration](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5F4340e6ac-b3c4-4bc8-b396-9cd1dbcb2515)

You can see what’s changed on your site by exporting a CSV report.

The audit log lists administrator details, what’s changed, and when the change occurred.

**_Audit Log_**

Report output varies depending upon your site configuration. To see which settings are typically included, see the [Webex Audit Log](http://www.cisco.com/c/dam/en/us/td/docs/collaboration/webex%5Fcenters/esp/Site%5FConfig%5FAudit%5FLog%5FListing.pdf).

| 1 | Sign in to Control Hub at [https:/​/​admin.webex.com](https://admin.webex.com), and open the Troubleshooting page.                                                                          |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Click the Admin Activities tab. You can filter by specific administrators, event descriptions, or impacted resources. You can also reduce the date range to help you find specific events.  |
| 3 | Click Export to CSV to download the events to a file. For more information about the events in the activity log, see [Control Hub audit events reference](https://help.webex.com/nq4juobb). |

You can see what’s changed on your site by exporting a CSV report. The change log tracks the setting changes made on the  Configuration  \>  Common Site Settings  \>  Options  page.

The audit log lists administrator details, what’s changed, and when it happened.

**_Audit Log_**

Report output varies depending upon your site configuration. To see which settings are typically included, see the [ Webex Audit Log ](http://www.cisco.com/c/dam/en/us/td/docs/collaboration/webex%5Fcenters/esp/Site%5FConfig%5FAudit%5FLog%5FListing.pdf).

| 1 | Sign in to Site Administration and go to  Reports  \>  Common  \>  Audit Log .                                                                                                                 |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the  Site Configuration  tab.                                                                                                                                                           |
| 3 | Use the  From  and  To  menus to select a date range, and then click ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/422001-423000/422055.jpg). |
| 4 | Choose to either open or save the file, and then click  OK .                                                                                                                                   |
