# Get started with Webex Meetings Suite

## Getting started with the Webex Meetings Suite

To get started with the Webex Meetings Suite, you need the following:

- A host account on your Webex site.
- The Webex URL for User Hub.
- Your Webex username and password.

The welcome email that you receive from your site administrator contains your Webex URL. Visit the URL to sign in to User Hub. From there, you can easily meet online, over the phone or your computer, with anyone anywhere in the world.

The welcome email that you receive may also include a link for you to create a password for your account. To create a password for your account, select Create Password.

If your Webex site uses Single-Sign-On (SSO), use those credentials to sign in.

## Sign in to User Hub

The welcome email that you receive contains the site URL, your username, and a link to set up your password. If your site uses SSO, you use those credentials to sign in to your Webex site.

| 1 | Go to User Hub in your browser.                                            |
| - | -------------------------------------------------------------------------- |
| 2 | Select Sign in.                                                            |
| 3 | At the prompt, enter your username and your password, then select Sign in. |

## Install Webex App

Webex App allows users to start and join the meetings quickly and easily includes the pre-meeting and in-meeting experience. Your site administrator can set Webex to automatically download the first time that you sign in. If not, we highly recommend that you download it.

| 1 | Go to <https://www.webex.com/downloads.html>                                                                                                                                                      |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Click the version that you want to download. If you want to install the Webex Meetings app, go to Our preview app, Meetings and select Download or choose the mobile app version for your device. |
| 3 | Open the folder where saved the installation file, and then double-click the installation file.                                                                                                   |
| 4 | Follow the on-screen prompts to install.                                                                                                                                                          |
