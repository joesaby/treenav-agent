# View Reports for Webex Training

## View reports for Webex Training

| 1 | Sign in to Webex Site Administration, go to Reports  \> Webex Training , and then select one of the following reports: Usage Report : View usage information for all sessions. Registration Report : View all registration information for all sessions scheduled and in the past. Recorded Session Access Report : View access for all recorded sessions. Recording Access Report : View access for all recorded sessions. Coupon Report : View the details of coupons that were issued and used. Training Report by Attendee : View a record of all sessions for which a person has registered or attended. Attendee Report : View a record of all sessions for which a person has registered or attended. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Specify the search criteria, and then click Display Report .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
