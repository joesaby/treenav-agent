# Known issues and limitations for people insights profiles

People insights profiles have the following known issues and limitations:

- In the Participants  panel in Meetings, Learn more about this participant  doesn't appear next to the names of people who join from Webex App, regardless of whether a profile is available to view.
- Admins can only edit company details by sending feedback using the in-app button or by submitting a ticket using the in-app Send Feedback  button.
- Choosing specific fields from Directory Synchronization to show in profiles doesn't work currently.
