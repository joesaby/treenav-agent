# Webex App | Access call statistics

| During a call, tap More ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/423001-424000/423879.jpg), and then select Show statistics. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/447001-448000/447159.jpg) BSCF Call —Your phone services are using Webex Calling. CUCM Call—Your phone services are using Unified CM. Blank indicator—You're on a Webex App call. For more information, see [Supported Calling Options](https://help.webex.com/xga73p/). |
