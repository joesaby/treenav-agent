# Emergency alarms on your DECT handset

## Emergency alarm list

You can view the list of emergency alarms you received, if emergency alarms are configured to ring on your handset.

## View the received emergency alarms

| 1 | Press Menu ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/390001-400000/393001-394000/393848.jpg).        |
| - | ------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select Alarm list ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/432001-433000/432091.jpg). |
| 3 | If there are alarms, press Select to view the alarm information.                                                                            |

## Report an emergency situation

Your 6825 Handset has an emergency button, located at the top of the handset. If set up by your administrator, you can press this button to raise an alert in an emergency situation. The handset places a call to a predefined emergency number and automatically turns on the speakerphone.

The 6823 Handset doesn't have an Emergency button.
