# Search and call contacts on your phone

## Browse contacts

You can browse through the contacts in your corporate directory and in your personal address book.

| 1 | Do one of these actions: Desk phones—Press Contacts ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381338.jpg)Conference phone—Press Contacts.                                                                                                                                                                                                                                                                                     |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Press Category, if it displays. Otherwise, skip the step.                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| 3 | Do one of these actions: Select Personal address book. Press the up or down arrows on the navigation pad to scroll through the contacts in your directory. Or, select Option \> Search and then enter the address entry to search for. The matched contacts will display on the phone.Select All directories and then enter the address entry to search for. The matched contacts will display on the phone.. You can only search by name (case-insensitive). Search by number is not supported. |

## Call a contact in your personal address book

Before you begin

Your administrator must enable the personal directory feature on the phone. Otherwise, the Personal address book menu doesn't display.

| 1 | Press Contacts ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381338.jpg). |
| - | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Press Category, if it displays. Otherwise, skip the step.                                                                                |
| 3 | Select Personal address book and search for an entry.                                                                                    |
| 4 | Highlight the personal address book entry that you want to dial.                                                                         |
| 5 | Press Call.                                                                                                                              |

## Edit a contact in your personal address book

Before you begin

Your administrator must enable the personal directory feature on the phone. Otherwise, the Personal address book menu doesn't display.

| 1 | Press Contacts ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/380001-390000/381001-382000/381338.jpg).                                                                 |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Press Category, if it displays on the phone. Otherwise, skip the step.                                                                                                                                   |
| 3 | Select Personal address book and search for an entry.                                                                                                                                                    |
| 4 | Highlight the entry you want to change.                                                                                                                                                                  |
| 5 | Do one of these actions: Conference phone: Press Edit.Desk phone: Press Select ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/370001-380000/371001-372000/371856.jpg). |
| 6 | Modify the entry information.                                                                                                                                                                            |
| 7 | Press Save.                                                                                                                                                                                              |
