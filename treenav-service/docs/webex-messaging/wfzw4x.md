# Webex App | Edit your people insights profile

| 1 | Click your profile picture, then click Edit Profile, and select  View People Insights Profile.                                                                     |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | In the  People Insights Profile  window that appears, click  Edit profile. If you want to hide the information that people insights provides, click  Hide profile. |
| 3 | Add your information to edit your profile. You can also create or edit your profile from [ https://people.webex.com ](https://people.webex.com).                   |
