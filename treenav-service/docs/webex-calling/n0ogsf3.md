# Delete a File from Your Media Manager

| 1 | From the customer view in <https://admin.webex.com>, go to Services \> Call \> Settings, scroll to Company Media On Hold, and then click Media Manager.                                     |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Choose the file you want to delete, click ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/420001-421000/420184.jpg), and then choose Delete. |
| 3 | (Optional) Go to your trash and delete all files. Click Delete All.                                                                                                                         |
