# Delete or restore a Webex recording

The Delete button is only available in User Hub if the administrator has turned it on in Control Hub.

| 1 | Sign in to [User Hub](https://globalpage-prod.webex.com/signin).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Select Recordings from the navigation panel. On the Recordings tab, you can see recordings for the following: Webex Meetings- These recordings include meetings, events, and training sessions that you hosted and recorded. You can keep up to 900 GB of recordings. Calling- These recordings include all the calls you made.                                                                                                                                                                                                                  |
| 3 | Select Delete ![Trash,](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/431001-432000/431295.jpg) next to the name of the recording, and then select Delete. To delete multiple recordings, check the box next to multiple recordings and then select Delete \> Delete. For Webex Calling, if your administrator provides you with the delete access, then you see the Delete button. The recordings move to the trash and appear on the Deleted tab for 30 days, unless you permanently delete them. |
| 4 | To delete a recording from the trash permanently, select Deletedfrom the recordings drop-down menu, then click Delete ![Trash,](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/431001-432000/431295.jpg) next to the name of the recording. To delete multiple recordings permanently, check the boxes next to multiple recordings and then select Delete \> Delete. To delete all recordings permanently, select  All \> Delete.                                                                    |
| 5 | To move a recording from the trash back to your recordings list, select Restore ![Restore,](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/431001-432000/431296.jpg) next to the name of the recording.                                                                                                                                                                                                                                                                                              |
