# Configure call intercept in Control Hub

- [Configure for a location](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5F05fa89ae-e329-43fc-9346-b26b70efb3d9)
- [Configure for a user](#id%5F136742)
- [Configure for a workspace](#id%5F137151)

You can intercept incoming or outgoing calls for a location. If a user or place already has intercept enabled, those settings override any location intercept settings that you configure.

| 1  | Sign in to [Control Hub](https://admin.webex.com/).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2  | Go to Management \> Locations. You can also find the Locations from Services \> Calling and from Calling quick links tile, select Locations.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 3  | Select the location that you’d like to configure for Location Intercept.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 4  | Go to Calling tab and click Call intercept under the Call handling section.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 5  | Toggle to enable Call Intercept.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| 6  | Under Incoming calls, choose Allow all incoming calls or Intercept all incoming calls.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 7  | If you choose Intercept all incoming calls, choose the features that you need. To send location calls to the voicemail of an individual user, check the Send all calls to voicemail check box. When choosing Announcements, select the default greeting, or upload a custom audio greeting from the drop-down list and then click Upload File. The audio (WAV) file must meet specified requirements.![Intercept all incoming calls](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/460001-470000/464001-465000/464853.png)                                                                                                                                                                                                                                                                                                                                                                               |
| 8  | Under Phone Number Announcements, choose from the following options, and enter the number redirects: Play new number announcements—This number is announced to the caller as the new number to call. This number can be an internal company number or extension, or a valid e164 external PSTN number. Transfer on '0' to this phone number—The caller is transferred to the number entered when 0 is pressed. If both of these options are checked, it’s possible to enter two different numbers for each option, but it results in a misleading announcement message.  Example, if the Play new number announcements are set to 1111 and the Transfer on '0' to this phone number to 2222, callers hear the message "The number you’re trying to reach is out of service. The new number is 1111\. Press 0 now if you wish to be transferred to the new location." In this case, pressing '0' sends callers to 2222 rather than 1111. |
| 9  | For Outgoing Calls, choose from the following: Intercept all outgoing calls—When the user attempts an outgoing call, they hear a message telling them their line is out of service. Allow only national outgoing calls—A user from the intercepted location will only be allowed to make national calls. Transfer intercepted calls to a phone number—When a user from the intercepted location makes any outgoing call, the call is transferred to a designated number.                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 10 | Click Saveto confirm your changes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |

You can intercept incoming or outgoing calls for users in your organization. If this setting is enabled, calls are either routed to a designated number you choose, or to the user's voicemail.

| 1 | Sign in to [Control Hub](https://admin.webex.com/).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Go to Management \> Users and select the user that you want to edit.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| 3 | Go to Calling tab and click Call Intercept section.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| 4 | Toggle the button to change the settings for the user.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| 5 | Under Incoming Calls, choose Allow all incoming calls or Intercept all incoming calls. ![Choose Intercept all incoming calls for a user](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/448001-449000/448783.png)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 6 | If you choose Intercept all incoming calls, set up the features you need: To send users calls to voicemail, check the Send all calls to voicemail check box. When choosing Announcements, select the default greeting, or upload a custom audio greeting from the drop-down list, and then click Upload File.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 7 | Under Phone Number Announcements, choose from the following options, and enter the number redirects: Play new number announcements—This number is announced to the caller as the new number to call. Transfer on '0' to this phone number—The caller is transferred to the number entered when 0 is pressed. If both of these options are checked, it’s possible to enter two different numbers for each option, but it results in a misleading announcement message. For example, if the Play new number announcements is set to 1111 and the Transfer on '0' to this phone number to 2222, callers hear the message _"The number you’re trying to reach is out of service. The new number is 1111\. Press 0 now if your wish to be transferred to the new location."_ In this case, pressing '0' sends callers to 2222 rather than 1111. |
| 8 | For Outgoing Calls, choose from the following: Intercept all outgoing calls—When the user attempts an outgoing call, they hear a message that this line is out of service. Allow only local outgoing calls—The user is allowed to make only national calls. Transfer intercepted calls to a phone number—Any outgoing call that the intercepted user makes is transferred to a designated number.                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| 9 | Click Save to confirm your changes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

You can intercept incoming or outgoing calls for workspaces in your organization. If this feature is enabled, calls are either routed to a designated number you choose, or to the workspace's voicemail.

To make this feature available, a workspace license must be assigned to the workspace.

| 1 | Sign in to [Control Hub](https://admin.webex.com/).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Go to Management \> Workspaces and select the workspace that you want to edit.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 3 | Select Calling.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 4 | In the Call handling card, select Call intercept.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| 5 | Under Incoming calls and Outgoing calls the call intercept functionality can be configured once Call intercept is enabled. ![Configuring incoming call settings under Workspaces in Control Hub.](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/490001-500000/494001-495000/494449.png)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 6 | If you choose Intercept all incoming calls, set up the features you need: To send workspace calls to voicemail, check the Send all calls to voicemail check box. When choosing Announcements, select the default greeting, or upload a custom audio greeting from the drop-down list, and then click Upload File.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| 7 | Under Phone Number Announcements, choose from the following options, and enter the number redirects: Play new number announcements—This is the number that will be announced to the caller as the new number to call. Allow only national outgoing calls—The workspace is allowed to make only national calls. Transfer intercepted calls to a phone number—Any outgoing call that the intercepted workspace makes is transferred to a designated number.                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| 9 | Click Save to confirm your changes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
