# Create Cisco Webex Events Registration Form Templates in Cisco Webex Site Administration

## Create a Registration Form Template

Hosts can choose to use registration form templates that you create. If you don't create a template, then hosts can only use the basic default template.

You can also allow hosts to use their own [customized form](https://help.webex.com/article/n536yej).

| 1 | Sign in to Webex Site Administration, and go to Configuration \> Webex Events \> Registration Form. |
| - | --------------------------------------------------------------------------------------------------- |
| 2 | Select Template Form, and then click Create Template.                                               |
| 3 | Click Save.                                                                                         |

## Manage Registration Form Templates

You can modify, enable, disable, or delete a registration form template.

| 1 | Sign in to Webex Site Administration, and go to Configuration \> Webex Events \> Registration Form. |
| - | --------------------------------------------------------------------------------------------------- |
| 2 | Click the ellipsis … icon next to a template, and select the action that you want to perform.       |
| 3 | Click Save.                                                                                         |
