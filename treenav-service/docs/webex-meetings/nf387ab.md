# Check the Webex Productivity Tools Version for Windows

| 1 | To check the Webex Productivity Tools Version, do one of the following: In Microsoft Outlook, go to Schedule Meeting  and select About  from the menu. Go to your Webex site and select Downloads  \> Version information . |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | If your installed version is not the latest version listed on your site, go to the Downloads page of your Webex site and click Download  under Cisco Webex Productivity Tools.                                              |
