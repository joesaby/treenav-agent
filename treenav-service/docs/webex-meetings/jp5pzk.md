# Manage your Webex Meeting using the keypad on your phone

| Meeting Control                                                  | Press |
| ---------------------------------------------------------------- | ----- |
| Mute or unmute                                                   | \*6   |
| Lock or unlock (available only for Webex Personal Room meetings) | \*5   |
| Start recording                                                  | \*9   |
| Stop recording                                                   | #9    |
| Mute all                                                         | ##    |
| Unmute all                                                       | 99    |
