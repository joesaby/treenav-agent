# Using SCORM in Webex Training

## SCORM overview

In SCORM pages, the learning management system (LMS) can track what needs to be delivered to the learner, when the learner achieves the skill or competency, and sends the learner to the appropriate level of content. 

An Application Programming Interface (API) provides a standard way of communicating with the LMS using Web-friendly JavaScript. Webex Training provides a SCORM page in which the host can specify which information is provided in the API. 

You can read more about SCORM at the following websites: 

- www.adlnet.org
- www.imsglobal.org
- www.rhassociates.com/scorm.htm
- www.teleologic.net/SCORM/index.htm
- www.altrc.org/specifications.asp

## View or edit SCORM data

| 1 | Sign in to your Webex site and go to your upcoming meetings.                                            |
| - | ------------------------------------------------------------------------------------------------------- |
| 2 | Select the link for the training session that you want to view or edit.                                 |
| 3 | Select a link next to SCORM data.                                                                       |
| 4 | If necessary, change the information in the text boxes and drop-down lists, and then save your changes. |
| 5 | To download the file, select Save and Download.                                                         |
