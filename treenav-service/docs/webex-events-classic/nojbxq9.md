# Show Custom Toll-Free Dialing Restrictions

Cloud Connected Audio (CCA) customers and partners can provide a custom _Toll Free Calling Restrictions_ document.

| 1 | Sign in to Webex Site Administration and go to Configuration  \> Common Site Settings  \> Options .                           |
| - | ----------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the Site Options  section, from the Toll-free dialing restrictions  list, select Customer toll-free dialing restrictions . |
| 3 | To show customer toll-free dialing restrictions, select Yes .                                                                 |
| 4 | To provide the document on a web page, select URL , and then enter the URL in the corresponding field.                        |
| 5 | To provide a PDF version of the document, select PDF file , and then click Upload File .                                      |
| 6 | Browse to the file location, select the file, and then click Open .                                                           |
| 7 | Select Update .                                                                                                               |
