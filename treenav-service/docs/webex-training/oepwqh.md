# Manage Your Settings in Cisco Webex Training

## Update Training Site Settings

| 1 | Select Configure \> Webex Training \> Options.                                                                                               |
| - | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Specify options under Site Options. [See this article if you're setting up E-Commerce.](https://collaborationhelp.cisco.com/article/ns9kodx) |
| 3 | Select Update.                                                                                                                               |

## Manage Your Settings in Cisco Webex Training

| Option                             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Number of recurrences allowed:** | You can enter the max number of recurring single-session class or multiple-session course. The default is 30.                                                                                                                                                                                                                                                                                                                                                            |
| **Join training options**          | **Attendees can join using Flash**: Specifies if you want to allow attendees to join using Flash technology to join a training. Select if you want Flash to be tried **First** or **Last**.                                                                                                                                                                                                                                                                              |
| **Default Email Options**          | You can set different email options so users are notified about training updates, registration, and more.                                                                                                                                                                                                                                                                                                                                                                |
| **In-event options**               | **Automatically record**: Specifies if you want to automatically record all sessions using Network Based Recording (NBR). **Send notification email**: Specifies if you want an email sent to the host when the training recording is ready. **Enable attention tracking**: Specify if you want to enable the attention tracking functionality for meetings. The **Automatically record** feature is only available if the recording option is provisioned on your site. |
| **Default Options**                | Select the Default Page for users to view. Options are applied to the site as defaults, but individual users can change them.                                                                                                                                                                                                                                                                                                                                            |
| **Default Attendee landing page**  | Specifies page to display to meeting attendees when meetings end. Choose from the following options: **Use Webex branded default page:** To display the default Webex page. To also display promotional content for free Webex trials, select **Show promotional content**. **Use customized page:** Enter the URL of another page to display to attendees when meetings end. You do not need not enter "http://" in the URL.                                            |
| **Default host landing page**      | Specifies page to display to meeting hosts when meetings end. Choose from the following options: **Use Webex branded default page:** To display the default Webex page. To also display promotional content for free Webex trials, select **Show promotional content**. **Use customized page:** Enter the URL of another page to display to hosts when meetings end. You do not need not enter "http://" in the URL.                                                    |
| **Invitation email**               | To display promotional content for free Webex trials on invitation emails, select **Show promotional content**.                                                                                                                                                                                                                                                                                                                                                          |
| **Join page**                      | To display promotional content for free Webex trials on the page that hosts and attendees use to join meetings, select **Show promotional content**.                                                                                                                                                                                                                                                                                                                     |
