# Manage Session Types

After the initial configuration, you’ll want to [manage session types](https://collaborationhelp.cisco.com/article/acyd33). Do so before you import users because you can set a [default session type to be added to all new users](https://collaborationhelp.cisco.com/article/acyd33#ID%5F2473%5F00001b88), which is easier than applying a session type to each user after they’re imported.
