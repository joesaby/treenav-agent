# Webex App | Choose a Webex device to use

## Devices on your network

Your administrator can configure devices on your network in various ways. You can have on-premises devices or cloud devices. These devices can appear the same and you can connect to them using ultrasound.

## Connecting to devices

- On-premises devices—You can connect to a device using ultrasound. For more information, see [ Automatically connect to a Webex device](https://help.webex.com/article/8zp0oz) or [Manually connect to a Webex device](https://help.webex.com/article/nf29igm/).
- Cloud devices—You can connect to a device using ultrasound. For more information, see [ Automatically connect to a Webex device](https://help.webex.com/article/8zp0oz). You can also connect using the following:

  - Search for devices that are nearby and choose a device to connect to. For more information, see [ Find and connect to nearby Webex devices](https://help.webex.com/article/qtsdfy).
  - Search for devices and manually connect to them using a PIN. For more information, see [ Manually connect to Webex devices](https://help.webex.com/article/nf29igm).

## Features for connected devices

When you connect to a device, you can use it in the following ways:

- Share your screen locally, you can only share audio and optimize video when you use a cloud device. For more information, see [ Webex App | Share content wirelessly to Webex devices ](https://help.webex.com/article/pceb9t).
- Call somebody using the device and you can use Webex App to control the device. For more information, see [Webex App | Supported calling options](https://help.webex.com/article/ge0ybp).
- Answer a call from somebody using the device and you can use Webex App to control the device. For more information, see [Webex App | Answer a call](https://help.webex.com/article/n19ix1l).  
Some Webex devices that you connect to may not be enabled for calling. If you try to make a call while connected to this kind of device, you'll see an error message in Webex App telling you to disconnect and use another device or your app to make the call instead.
- Start a meeting in a Webex App space using the device and you can use Webex App to control the device. For more information, see [Webex App | Start a meeting from a space ](https://help.webex.com/article/nyrqyyw).
