# Register for an Event in Webex Events (classic)

## Register for One or More Events

Before you begin

Register for the Webex event that you want to attend. If the events are part of a program, you can register for multiple events simultaneously.

| 1 | Sign in to your Webex site, and select **Webex Events (classic)**  in the left panel.                                                                                                                                                               |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the title of the event or program and click Register.                                                                                                                                                                                        |
| 3 | If a password is required, enter it, then click Submit. You can find the password in the event invitation.                                                                                                                                          |
| 4 | On the registration page, enter the required information, then click Submit.                                                                                                                                                                        |
| 5 | (Optional) If the Payment Information page appears, enter any applicable promotion codes.                                                                                                                                                           |
| 6 | Provide the required payment information if there is a balance, then click Submit. If the host required a registration ID for the event, be sure to keep the registration confirmation email. You will need to enter the ID when joining the event. |

## Change the Locale, Language, or Time Zone for Your Site

You can change your Webex site preferences at any time, including the locale, language, and time zone. If you have a user account, you can also change these settings in the general section of your personal preferences.

When you sign in to the Webex site from your mobile device's browser, the time zone you see will match what you've set in Preferences. If you're not signed in, the time zone will match what's set for the Webex site.

| 1 | Sign in to your Webex site, and select Webex Events (classic)in the left panel.                                                                                  |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Go to Set Up \> Preferences \> General. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/422001-423000/422218.jpg) |
| 3 | Make your changes and click Save.                                                                                                                                |
