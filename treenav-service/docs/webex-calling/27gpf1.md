# Executive and Executive Assistant Feature Access Codes

| Feature Access Code Name              | Default Value | Associated Service  |
| ------------------------------------- | ------------- | ------------------- |
|                                       |               |                     |
| Executive Call Filtering Activation   | #61           | Executive           |
| Executive Call Filtering Deactivation | #62           | Executive           |
| Executive-Assistant Call Push\*       | #63           | Executive Assistant |
| Executive-Assistant Initiate Call\*   | #64           | Executive Assistant |
| Executive Assistant Opt-In\*          | #65           | Executive Assistant |
| Executive Assistant Opt-Out\*         | #66           | Executive Assistant |

\*Requires additional digits when dialed.

## Feature Access Code Requiring Additional Digits

The following is an example of the dialing sequence options for using an executive-assistant FAC.

**Opt-in feature access code example:** 

If the executive's phone number or extension is not provided with the FAC, then the assistant is prompted for the executive's phone number or extension.

- #65 - Then prompted for the executive's phone number or extension.
- #65 + phone number or extension.
