# Send Email Messages to All Webex site Users in Control Hub

## 

| 1 | Sign in to [Control Hub](https://admin.webex.com).                                                                                                                                 |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | In the left navigation pane under Services, click Meeting.                                                                                                                         |
| 3 | Select the Webex site that you want send the emails to.                                                                                                                            |
| 4 | Select Email All Hosts, write your message, then select  Send . In your message to all users, do not include any variables that are in the email templates for your Webex service. |
