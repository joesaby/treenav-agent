# Make a conference call from your analog phone

You can talk with several people in a single call.

| 1 | While on an active call, perform a hook flash. This puts the call on hold and you hear dial tone. |
| - | ------------------------------------------------------------------------------------------------- |
| 2 | Dial a second number and wait until that person answers.                                          |
| 3 | To create the conference, perform a hook flash. You are now in a conference with both persons.    |
