# Answer calls on your desk phone

## Answer a call

| Do one of these actions: 6800, 7800—Press the flashing red line button. 8800—Press the flashing red session button. If you have multiple lines on your phone, press the solid red line button first.If Auto Collapse Into Line Key parameter is enabled, the active call window is minimized and the call session information such as name of the caller, number, call duration, call state, and any special icons like secure call, call recording, etc. are displayed in the line label itself.During multiple calls, the line icon is switched between call counts and current call icon. To show full screen call window, press the line key. |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

## Answer call waiting

When you're on an active call, you know that a call is waiting when you hear a single beep and the session or line button flashes red.

| Do one of these actions: 6800, 7800—If you have more than one call waiting, select an incoming call. Press the line button.8800—Press the session button. If you have more than one call waiting, select an incoming call. |
