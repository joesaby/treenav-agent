# Change Language Settings for a Cisco Webex Site in Cisco Webex Control Hub

## Change the Default Language for a Site

You can change the default language of your Webex site, so that users who haven’t set their language preference see the site and meeting invitations in the default language.

| 1 | Sign in to [Control Hub](https://admin.webex.com), and go to Services \>  Meeting.                                 |
| - | ------------------------------------------------------------------------------------------------------------------ |
| 2 | Click Sites, choose the Webex site that you want to change the settings for, and then click Settings.              |
| 3 | Under Common Settings, select Site Options.                                                                        |
| 4 | In the Languages section, choose the default language from the Default language for the Webex site drop-down list. |
| 5 | Click Save.                                                                                                        |

## Change the Languages Available to Users

You can choose what languages are available to your users, in case your site does not support certain languages.

| 1 | Sign in to [Control Hub](https://admin.webex.com), and go to Services \>  Meeting.                                           |
| - | ---------------------------------------------------------------------------------------------------------------------------- |
| 2 | Click Sites, choose the Webex site that you want to change the settings for, and then click Settings.                        |
| 3 | Under Common Settings, select Site Options.                                                                                  |
| 4 | Go to Languages available in Preferences, and check the check box next to each language you want to make available to users. |
| 5 | Click Save.                                                                                                                  |
