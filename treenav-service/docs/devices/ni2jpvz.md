# Hold Calls with Your Cisco Headset 500 Series

- [520 and 530 Series](#reference%5F61E539B3533CF716A3D87699E08C9268)
- [560 Series](#reference%5FD57AD3B8FF0027DEE3C02EC5B47A57C5)

[Place a Call on Hold and Resume a Held Call](#task%5F891073DE2174990B8ADFF5701E59BE0F)

| 1 | Press Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/425001-426000/425658.jpg) on the inline controller. |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the call you want to resume.                                                                                                                                          |
| 3 | Press Call.                                                                                                                                                                  |

[Put a Call on Hold and Answer an Incoming Call](#task%5FB61ACCC31A96D1ABB5FE4BA4F750CB76)

| Press ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/425001-426000/425658.jpg) once. |
| --------------------------------------------------------------------------------------------------------------------------------------------------- |

[End a Call and Resume a Held Call](#task%5F0F2DE1B887392771F2A84A24642CB90B)

| 1 | Hold Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/425001-426000/425658.jpg) until you hear a tone. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | On your phone, select the call that you want to resume.                                                                                                                  |
| 3 | Press Call once.                                                                                                                                                         |

[End a Call and Answer an Incoming Call](#task%5F4586FE74F69C940D987DAAD3E317262B)

| 1 | Hold Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/425001-426000/425658.jpg) until you hear a tone. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Press Call once.                                                                                                                                                         |

[Put a Call on Hold and Resume a Call](#task%5F0EB44F53637056286E4DB6A31E71DCB4)

| 1 | Press Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/390001-400000/393001-394000/393941.jpg) on your headset. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the call that you want to resume.                                                                                                                            |
| 3 | Press Call.                                                                                                                                                         |

[Hold a Call and Answer an Incoming Call](#task%5F5158F338ED29C8B6F3EF240300F221E2)

| Press ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/390001-400000/393001-394000/393941.jpg) once. This call feature is unavailable to when connected to a Cisco IP Phone with Firmware 12.1(1). |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

[End a Call and Resume a Held Call](#task%5FAF6490FB6EB801597DE530340BD5D900)

| 1 | Hold Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/390001-400000/393001-394000/393941.jpg) on your headset until you hear a tone. |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the call that you want to resume.                                                                                                                                                 |
| 3 | Press Call.                                                                                                                                                                              |

[End an Active Call and Answer an Incoming Call](#task%5F4D9CA8508211FC9FD1AA52766F062D41)

| 1 | Hold Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/390001-400000/393001-394000/393941.jpg) until you hear a tone. |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 2 | Press Call ![the call button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/300001-400000/390001-400000/393001-394000/393941.jpg) once.                 |
