# Reassign a User's Recordings to Another Webex Account

| 1 | Sign in to Webex Site Administration and go to User Management  \> Edit User .                                                                                                                                                  |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Locate the user account and select the link next to their user ID.                                                                                                                                                              |
| 3 | Scroll down to the Recordings and Content Management  section.                                                                                                                                                                  |
| 4 | Check the Recordings  check box, and select Reassign .                                                                                                                                                                          |
| 5 | On the Reassign Owner  page, search for and select the new owner of the recordings, and select OK . To access the recordings, the new owner must sign in to the Webex site, and go to My Webex  \> My Files  \> My Recordings . |
