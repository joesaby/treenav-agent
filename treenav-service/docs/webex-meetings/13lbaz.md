# Upgrade Multiple Webex Sites for Customers with Batch Site Upgrade in Control Hub

- [Schedule Upgrade](#id%5F122104)
- [History](#id%5F122105)

You can upgrade up to 150 Webex sites in one CSV file. You can schedule a batch site upgrade for:

- Webex sites that have an active license.
- Webex sites that aren't part of a trial.
- Webex sites that you manage for your customers.

You can cancel scheduled batch site upgrades, download a report of batch site upgrades that have finished, or view the status of the batch site upgrade in the History tab.

## Schedule a Batch Site Upgrade

## Schedule a Batch Site Upgrade

You can't roll back site version upgrades.

| 1 | From the partner view in <https://admin.webex.com>, go to Services, and under Meeting, select Batch Site Upgrade.                                                     |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Choose: Now to upgrade the Webex sites right away. Future Time to schedule a batch site upgrade. Choose a date and enter the time that you want the upgrade to start. |
| 3 | Upload a CSV file with the Webex sites that you want to upgrade. You can click Validate Sites to make sure your CSV file has the correct Webex site names.            |
| 4 | Select Schedule Upgrade.                                                                                                                                              |

You can view a detailed list of batch site upgrades that have been scheduled for your customers. The list displays who started the batch site upgrade, the start and end time of a batch site upgrade, and the actions that you can perform on a batch site upgrade.

## View the Status of a Batch Site Upgrade

## View the Status of a Batch Site Upgrade

You can view the status of a batch site upgrade that you scheduled within the last six months.

| 1 | From the partner view in <https://admin.webex.com>, go to Services, and under Meeting, select Site Upgrade History. |
| - | ------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Status, you can view if a batch site upgrade was successful or not.                                           |

## Download a Report of a Batch Site Upgrade

## Download a Report of a Batch Site Upgrade

You can download a report that shows you the status of each Webex site in the batch site upgrade.

| 1 | From the partner view in <https://admin.webex.com>, go to Services, and under Meeting, select Site Upgrade History.                                                                                                                    |
| - | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Actions, click the download ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/437001-438000/437210.jpg) button next to the batch site upgrade that you want to view the report for. |

## Cancel a Scheduled Batch Site Upgrade

## Cancel a Scheduled Batch Site Upgrade

You can only cancel a scheduled batch site upgrade that hasn't started yet.

| 1 | From the partner view in <https://admin.webex.com>, go to Services, and under Meeting, select Site Upgrade History.                                                                                       |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Actions, click the cancel ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/439001-440000/439614.jpg) button to cancel a scheduled batch site upgrade. |
