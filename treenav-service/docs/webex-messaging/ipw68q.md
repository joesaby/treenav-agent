| When reacting to a message, you can select an emoji from the emoji library, or use the quick-action emojis. Hover on a message and click ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/438001-439000/438739.jpg), then select an emoji reaction from the emoji library. Choose from different categories, select color tones, and search for an emoji using the search box.Hover on a message and select one of the displayed emojis in the action menu. When you add a reaction, it appears below the original message, if you want to remove your reaction just click on it again. You can add three reactions to any message, and if you hover over a reaction, you can see who added it. Webex App 43.6 is the minimum supported version for displaying reactions from the new emoji library. |

When you add a reaction, it appears below the original message, if you want to remove your reaction just tap it again. |
