# Hide meeting links from attendees within meetings on your Webex site

- [Site Administration](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5Fb14cf8de-135d-48cc-b7a9-f2a8de209388)
- [All meetings](#Cisco%5FTask%5Fin%5FList%5FGUI.dita%5F9c533902-7917-49fd-a0cc-6440d8450f35)

| 1 | Sign into Webex Site Administration, and go to Configuration \> Common Site Settings \> Options.                                                                                                                                                                                                                                                           |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Scroll down to Security Options \> Other and check Hide meeting link from attendee view within meetings (Meetings and Events). This option is unchecked by default. When hidden, Copy Meeting Link is disabled for attendees in the Meeting Info window, the More Options menu, and the Meeting menu. Hosts can still share meeting links within meetings. |

Hiding the links within meetings deters attendees from inviting unwanted guests by making the links less convenient to copy and share. It doesn’t prevent attendees from copying and sharing meeting links from their email invitations.

| 1 | Sign in to [Control Hub](https://admin.webex.com/), then go to Services \> Meeting.                                                                                                                                                                                                                                                                  |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Select the Webex site for which you want to change the settings, and select Configure Site.                                                                                                                                                                                                                                                          |
| 3 | Under Common Settings, select Security.                                                                                                                                                                                                                                                                                                              |
| 4 | Scroll down to Other and check Hide meeting link from attendee view within meetings (Meetings and Events). This option is unchecked by default. When hidden, the Copy Meeting Link option appears dimmed for attendees in the Meeting Info window, the More Options menu, and the Meeting menu. Hosts can still share meeting links within meetings. |
