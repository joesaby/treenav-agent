# Manage options in Webex Meetings and Webex Webinars

- [Windows](#Cisco%5FGeneric%5FTopic.dita%5F60ff32f9-4bcf-4e92-a50b-a1aaeada3c48)
- [Mac](#Cisco%5FGeneric%5FTopic.dita%5Ff0de4fbb-babb-49e8-8269-43dbc0a22c34)

## Windows

| 1 | In a meeting or webinar, click Show Menu Bar, click Meeting or Webinar and then click Options.      |
| - | --------------------------------------------------------------------------------------------------- |
| 2 | Select General, Content Sharing, or Import Mode.                                                    |
| 3 | Uncheck or check the options that you want to turn on or off, and click Apply to save your changes. |

## Mac

| 1 | In your meeting or webinar, click Webex Meetings or Webex Webinars and then click Meeting Options or Webinar Options. |
| - | --------------------------------------------------------------------------------------------------------------------- |
| 2 | Uncheck or check the options that you want to turn on or off, and click Apply to save your changes.                   |

|                 |                                                                                                                                                                                                                                                                           |                 |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------- |

| Content sharing | Revert to classic screen sharing                                                                                                                                                                                                                                          | Windows only    |
| Import mode     | Universal Communications Format Printer driver                                                                                                                                                                                                                            | Windows only    |
