# Reassign an Event to a Different Host in Control Hub

You can reassign an event to another Events (classic) host. The new host can edit the event to change information about it, including the password. 

| 1 | From the customer view in [https://admin.webex.com](http://admin.webex.com), go to  Services.                                                                                                                               |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Meeting, select Sites, choose your site, and then select Configure Site.                                                                                                                                              |
| 3 | Under Webex Events (classic), select Reassignment.                                                                                                                                                                          |
| 4 | Find the event in the list that appears. Alternatively, search for the event by selecting  Username, Event number, or Event date and then by entering the appropriate criteria in the Keywords box.                         |
| 5 | Click the Reassign link for the event that you want to reassign.                                                                                                                                                            |
| 6 | Find the new host in the list that appears. Alternatively, search for the event by selecting  Username, Name, Email address, or Phone Number. Then enter the appropriate criteria in the Keywords field to refine the list. |
| 7 | Select the new host and then click  Reassign.                                                                                                                                                                               |
