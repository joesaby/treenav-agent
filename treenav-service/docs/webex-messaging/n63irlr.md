# Webex App | Change the default location where files are saved

| 1 | Click Settings ![Settings button represented by a cog icon](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/420001-430000/427001-428000/427579.jpg) on the [navigation sidebar](https://help.webex.com/nx3i8h5/), and then click General. |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Under Default file location, click Change, select a new location to store files, and then click Select Folder.                                                                                                                                                         |
| 3 | On Windows, click Save.                                                                                                                                                                                                                                                |
