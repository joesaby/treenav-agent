# Search the Participants List in a Cisco Webex Meeting

- [Desktop](#task%5FE384AE4CEE5FD1ECF239D14E52862D33)
- [Mobile](#id%5F136654)

| 1 | Select ![Participants.](../../../../../../en/us/td/i/400001-500000/430001-440000/432001-433000/432174.JPG) to open the Participants panel.                                                                                                           |
| - | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Start typing a name in the Search bar. You can enter the name of a participant, or enter the name of a video device. When multiple participants connect to the same device to join the meeting, their names appear in a group below the device name. |
| 3 | To clear the Search bar, select ![Clear this entry button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/442001-443000/442281.jpg).                                                                     |

| 1 | Open the Webex Meetings app on your mobile device and tap Sign In ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/430001-440000/439001-440000/439272.jpg).                                                                                                                                                                                   |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Tap Join Meeting, enter the meeting number, Personal Room URL, or host's username, and then tap Join.                                                                                                                                                                                                                                                                         |
| 3 | Select ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/440001-441000/440464.jpg) to open the Participants panel.                                                                                                                                                                                                               |
| 4 | Start typing a name in the Search bar. You can enter the name of a participant, or enter the name of a video device. When multiple participants connect to the same device to join the meeting, their names appear in a group below the device name. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/447001-448000/447576.jpg) |
| 5 | To clear the Search bar, select ![Clear this entry button](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/442001-443000/442281.jpg).                                                                                                                                                                                              |
