# Create a remote support key for a Board, Desk, or Room Series device

| 1 | From the customer view in [https:/​/​admin.webex.com ](https://admin.webex.com), go to the Devices  page, and select your device in the list.                       |
| - | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Navigate to Support  and click Remote Access Key .                                                                                                                  |
| 3 | Click Copy to Clipboard  to copy the key. Share this key with Cisco Support by pasting it in a message.                                                             |
| 4 | You can deactivate the key after the support process. Click again on Remote Support Key  and Reset Key . This restricts the access that support has to this device. |
