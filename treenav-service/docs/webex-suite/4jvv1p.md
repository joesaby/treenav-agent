# Webex App | Choose as your Calling app

- [Windows](#id%5F132175)
- [Mac](#id%5F132176)

Make these changes to the system settings in Windows 10 to always make calls using Webex App.

| 1 | From the Windows search bar, search for Default app settingssystem settings, click Set defaults by app\>**Webex App** \> Manage. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/443001-444000/443295.jpg) |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | For each protocol in the list of associations that appears, choose Webex App. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/440001-450000/443001-444000/443294.jpg)                                                    |
| 3 | (Optional) Follow the steps in [Connect your Microsoft Outlook account](https://help.webex.com/yf1gc7/) if you want to make calls in Webex App when you click contact numbers in Microsoft Outlook for Windows.                                           |

What to do next

You can try to make a call from a number on a website or in Microsoft Outlook, and then the call takes place in Webex App. See [Click to call from another app](https://help.webex.com/n87vxwfb/).

On a Mac, you just need to make a change in Webex App.

| 1 | Click your profile picture, select Settings \> Calling.                                                                                                                                                     |
| - | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Click Start calls with and choose Webex App.                                                                                                                                                                |
| 3 | (Optional) Follow the steps in [Connect your Microsoft Outlook account](https://help.webex.com/yf1gc7/) if you want to make calls in Webex App when you click contact numbers in Microsoft Outlook for Mac. |
