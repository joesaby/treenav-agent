# Change Access Code or Phone Authentication Options for Webex Remote Access Computers

| 1 | Sign in to [Control Hub](https://admin.webex.com/) and go to Services \> Meeting.                              |
| - | -------------------------------------------------------------------------------------------------------------- |
| 2 | Select the site you want to access and select Settings \> Remote Access \> Groups.                             |
| 3 | Locate the computer for which you want to change options.                                                      |
| 4 | Select the link for the computer.                                                                              |
| 5 | In the Access code  or Phone authentication  section, specify the access code or phone authentication options. |
| 6 | Select Update  to save your changes.                                                                           |
