# Webex App | Turn off GPU acceleration for virtual backgrounds

GPU acceleration is supported on Windows and Intel GPUs that have a GPU driver version 26.20.0.0 or greater only.

| 1 | Click your [profile picture](https://help.webex.com/m67xht/), and then select Settings \> Video \> Advanced settings.                                                 |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Uncheck the Enable GPU acceleration for virtual background check box, and then click Save. If your system doesn't support this feature, you won't see this check box. |
