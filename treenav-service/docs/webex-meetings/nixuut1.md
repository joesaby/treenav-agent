# Screen sharing features on Webex devices

## Connect to a device

You can connect to a Webex device in different ways depending on which app you're using and how the device is connected to the network. 

|                                       | Webex App | Webex Meetings  |           |                 |
| ------------------------------------- | --------- | --------------- | --------- | --------------- |
|                                       | **Cloud** | **On-premises** | **Cloud** | **On-premises** |
| Ultrasound                            | ✓         | ✓               | ✓         | ✓               |
| Wi-Fi                                 | ✓         | —               | ✓         | —               |
| Search and connect manually using PIN | ✓         | —               | ✓         | —               |

## Local wireless sharing features

When you're in a physical room with everyone in the meeting, you can share you screen and applications on a Webex device. This way everyone in the room can see what you want to share on the big screen. 

|                     | Webex App    | Webex Meetings  |              |                 |
| ------------------- | ------------ | --------------- | ------------ | --------------- |
|                     | **Cloud**    | **On-premises** | **Cloud**    | **On-premises** |
| Optimized for text  | 1080p @3 fps | 1080p @3 fps    | 1080p @3 fps | 1080p @3 fps    |
| Optimized for video | 720p @30 fps | —               | 720p @30 fps | —               |
| Audio share         | ✓            | —               | ✓            | —               |
