# Activate Webex Cloud-Connected UC for Organizations

## Activate Webex Cloud-Connected UC For Organizations 

You can activate or enable Cloud-Connected UC for existing organizations from Control Hub in a seamless manner. This option is available only for customer organizations with on-premises Unified CM. It is not available for organizations with cloud-hosted Unified CM, such as Hosted Collaboration Solution (HCS) and UCM Cloud. 

Before you begin

Before you activate Cloud-Connected UC from Control Hub, make sure that you have full administrator privileges with Control Hub.

| 1 | From the customer view in [Control Hub](https://admin.webex.com/), go to Services \> Connected UC. ![](https://cisco-api.ingeniuxondemand.com/DITA/content/en/us/td/i/400001-500000/450001-460000/459001-460000/459152.jpg) |
| - | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 2 | Click Enable Connected UC, to activate the service on your account. The Connected UC page with UC Management and Analytics cards appear. You can go ahead and start to use Cloud-Connected UC services.                     |
